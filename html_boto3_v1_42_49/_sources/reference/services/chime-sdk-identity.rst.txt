======
Client
======



.. py:class:: ChimeSDKIdentity.Client

  A low-level client representing Amazon Chime SDK Identity
  

  The Amazon Chime SDK Identity APIs in this section allow software developers to create and manage unique instances of their messaging applications. These APIs provide the overarching framework for creating and sending messages. For more information about the identity APIs, refer to `Amazon Chime SDK identity <https://docs.aws.amazon.com/chime/latest/APIReference/API_Operations_Amazon_Chime_SDK_Identity.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('chime-sdk-identity')
