.. py:method:: TaxSettings.Client.delete_tax_registration(**kwargs)

  

  Deletes tax registration for a single account.

   

  .. note::

    

    This API operation can't be used to delete your tax registration in Brazil. Use the `Payment preferences <https://console.aws.amazon.com/billing/home#/paymentpreferences/paymentmethods>`__ page in the Billing and Cost Management console instead.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/taxsettings-2018-05-10/DeleteTaxRegistration>`_  


  **Request Syntax**
  ::

    response = client.delete_tax_registration(
        accountId='string'
    )
    
  :type accountId: string
  :param accountId: 

    Unique account identifier for the TRN information that needs to be deleted. If this isn't passed, the account ID corresponding to the credentials of the API caller will be used for this parameter.
