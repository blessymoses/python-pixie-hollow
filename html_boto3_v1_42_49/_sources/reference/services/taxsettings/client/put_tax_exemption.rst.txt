*****************
put_tax_exemption
*****************



.. py:method:: TaxSettings.Client.put_tax_exemption(**kwargs)

  

  Adds the tax exemption for a single account or all accounts listed in a consolidated billing family. The IAM action is ``tax:UpdateExemptions``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/taxsettings-2018-05-10/PutTaxExemption>`_  


  **Request Syntax**
  ::

    response = client.put_tax_exemption(
        accountIds=[
            'string',
        ],
        authority={
            'country': 'string',
            'state': 'string'
        },
        exemptionCertificate={
            'documentFile': b'bytes',
            'documentName': 'string'
        },
        exemptionType='string'
    )
    
  :type accountIds: list
  :param accountIds: **[REQUIRED]** 

    The list of unique account identifiers.

    

  
    - *(string) --* 

    

  :type authority: dict
  :param authority: **[REQUIRED]** 

    The address domain associate with the tax information.

    

  
    - **country** *(string) --* **[REQUIRED]** 

      The country code for the country that the address is in.

      

    
    - **state** *(string) --* 

      The state that the address is located.

      

    
  
  :type exemptionCertificate: dict
  :param exemptionCertificate: **[REQUIRED]** 

    The exemption certificate.

    

  
    - **documentFile** *(bytes) --* **[REQUIRED]** 

      The exemption certificate file content.

      

    
    - **documentName** *(string) --* **[REQUIRED]** 

      The exemption certificate file name.

      

    
  
  :type exemptionType: string
  :param exemptionType: **[REQUIRED]** 

    The exemption type. Use the supported tax exemption type description.
