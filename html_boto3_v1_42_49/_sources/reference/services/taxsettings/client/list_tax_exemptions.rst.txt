list_tax_exemptions
*******************



.. py:method:: TaxSettings.Client.list_tax_exemptions(**kwargs)

  

  Retrieves the tax exemption of accounts listed in a consolidated billing family. The IAM action is ``tax:GetExemptions``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/taxsettings-2018-05-10/ListTaxExemptions>`_  


  **Request Syntax**
  ::

    response = client.list_tax_exemptions(
        maxResults=123,
        nextToken='string'
    )
    
  :type maxResults: integer
  :param maxResults: 

    The number of results you want in one response.

    

  
  :type nextToken: string
  :param nextToken: 

    The token to retrieve the next set of results.

    

  
  
  :rtype: dict
  :returns:
