******************
list_connectors_v2
******************



.. py:method:: SecurityHub.Client.list_connectors_v2(**kwargs)

  

  Grants permission to retrieve a list of connectorsV2 and their metadata for the calling account.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/securityhub-2018-10-26/ListConnectorsV2>`_  


  **Request Syntax**
  ::

    response = client.list_connectors_v2(
        NextToken='string',
        MaxResults=123,
        ProviderName='JIRA_CLOUD'|'SERVICENOW',
        ConnectorStatus='CONNECTED'|'FAILED_TO_CONNECT'|'PENDING_CONFIGURATION'|'PENDING_AUTHORIZATION'
    )
    
  :type NextToken: string
  :param NextToken: 

    The pagination token per the Amazon Web Services Pagination standard

    

  
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of results to be returned.

    

  
  :type ProviderName: string
  :param ProviderName: 

    The name of the third-party provider.

    

  
  :type ConnectorStatus: string
  :param ConnectorStatus: 

    The status for the connectorV2.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

              The status for the connectorV2.
