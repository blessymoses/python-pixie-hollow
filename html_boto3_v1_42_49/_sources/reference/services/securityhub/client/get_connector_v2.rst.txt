get_connector_v2
****************



.. py:method:: SecurityHub.Client.get_connector_v2(**kwargs)

  

  Grants permission to retrieve details for a connectorV2 based on connector id.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/securityhub-2018-10-26/GetConnectorV2>`_  


  **Request Syntax**
  ::

    response = client.get_connector_v2(
        ConnectorId='string'
    )
    
  :type ConnectorId: string
  :param ConnectorId: **[REQUIRED]** 

    The UUID of the connectorV2 to identify connectorV2 resource.

    

  
  
  :rtype: dict
  :returns:
