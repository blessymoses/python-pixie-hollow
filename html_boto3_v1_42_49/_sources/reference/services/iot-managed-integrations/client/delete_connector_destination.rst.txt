delete_connector_destination

.. py:method:: ManagedintegrationsforIoTDeviceManagement.Client.delete_connector_destination(**kwargs)

  

  Delete a connector destination linked to a cloud-to-cloud (C2C) connector.

   

  .. note::

    

    Deletion can't be done if the account association has used this connector destination.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iot-managed-integrations-2025-03-03/DeleteConnectorDestination>`_  


  **Request Syntax**
  ::

    response = client.delete_connector_destination(
        Identifier='string'
    )
    
  :type Identifier: string
  :param Identifier: **[REQUIRED]** 

    The identifier of the connector destination.
