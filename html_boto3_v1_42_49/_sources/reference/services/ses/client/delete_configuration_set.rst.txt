************************
delete_configuration_set
************************



.. py:method:: SES.Client.delete_configuration_set(**kwargs)

  

  Deletes a configuration set. Configuration sets enable you to publish email sending events. For information about using configuration sets, see the `Amazon SES Developer Guide <https://docs.aws.amazon.com/ses/latest/dg/monitor-sending-activity.html>`__.

   

  You can execute this operation no more than once per second.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/email-2010-12-01/DeleteConfigurationSet>`_  


  **Request Syntax**
  ::

    response = client.delete_configuration_set(
        ConfigurationSetName='string'
    )
    
  :type ConfigurationSetName: string
  :param ConfigurationSetName: **[REQUIRED]** 

    The name of the configuration set to delete.

    

  
  
  :rtype: dict
  :returns:

      An empty element returned on a successful request.
