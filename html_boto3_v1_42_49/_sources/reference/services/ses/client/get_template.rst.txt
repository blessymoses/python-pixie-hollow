************
get_template
************



.. py:method:: SES.Client.get_template(**kwargs)

  

  Displays the template object (which includes the Subject line, HTML part and text part) for the template you specify.

   

  You can execute this operation no more than once per second.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/email-2010-12-01/GetTemplate>`_  


  **Request Syntax**
  ::

    response = client.get_template(
        TemplateName='string'
    )
    
  :type TemplateName: string
  :param TemplateName: **[REQUIRED]** 

    The name of the template to retrieve.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'Template': {
              'TemplateName': 'string',
              'SubjectPart': 'string',
              'TextPart': 'string',
              'HtmlPart': 'string'
          }
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **Template** *(dict) --* 

        The content of the email, composed of a subject line and either an HTML part or a text-only part.

        
        

        - **TemplateName** *(string) --* 

          The name of the template. You use this name when you send email using the ``SendTemplatedEmail`` or ``SendBulkTemplatedEmail`` operations.

          
        

        - **SubjectPart** *(string) --* 

          The subject line of the email.

          
        

        - **TextPart** *(string) --* 

          The email body that is visible to recipients whose email clients do not display HTML content.

          
        

        - **HtmlPart** *(string) --* 

          The HTML body of the email.

          
    
  
  **Exceptions**
  
  *   :py:class:`SES.Client.exceptions.TemplateDoesNotExistException`
