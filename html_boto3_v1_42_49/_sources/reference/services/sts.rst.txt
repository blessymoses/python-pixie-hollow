======
Client
======



.. py:class:: STS.Client

  A low-level client representing AWS Security Token Service (STS)
   

  Security Token Service (STS) enables you to request temporary, limited-privilege credentials for users. This guide provides descriptions of the STS API. For more information about using this service, see `Temporary Security Credentials <https://docs.aws.amazon.com/IAM/latest/UserGuide/id_credentials_temp.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('sts')
