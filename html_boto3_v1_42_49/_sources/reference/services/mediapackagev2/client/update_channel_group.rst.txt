update_channel_group
********************



.. py:method:: mediapackagev2.Client.update_channel_group(**kwargs)

  

  Update the specified channel group. You can edit the description on a channel group for easier identification later from the AWS Elemental MediaPackage console. You can't edit the name of the channel group.

   

  Any edits you make that impact the video output may not be reflected for a few minutes.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mediapackagev2-2022-12-25/UpdateChannelGroup>`_  


  **Request Syntax**
  ::

    response = client.update_channel_group(
        ChannelGroupName='string',
        ETag='string',
        Description='string'
    )
    
  :type ChannelGroupName: string
  :param ChannelGroupName: **[REQUIRED]** 

    The name that describes the channel group. The name is the primary identifier for the channel group, and must be unique for your account in the AWS Region.

    

  
  :type ETag: string
  :param ETag: 

    The expected current Entity Tag (ETag) for the resource. If the specified ETag does not match the resource's current entity tag, the update request will be rejected.

    

  
  :type Description: string
  :param Description: 

    Any descriptive information that you want to add to the channel group for future identification purposes.

    

  
  
  :rtype: dict
  :returns:
