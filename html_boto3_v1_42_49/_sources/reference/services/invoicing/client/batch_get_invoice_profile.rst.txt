*************************
batch_get_invoice_profile
*************************



.. py:method:: Invoicing.Client.batch_get_invoice_profile(**kwargs)

  

  This gets the invoice profile associated with a set of accounts. The accounts must be linked accounts under the requester management account organization.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/invoicing-2024-12-01/BatchGetInvoiceProfile>`_  


  **Request Syntax**
  ::

    response = client.batch_get_invoice_profile(
        AccountIds=[
            'string',
        ]
    )
    
  :type AccountIds: list
  :param AccountIds: **[REQUIRED]** 

    Retrieves the corresponding invoice profile data for these account IDs.

    

  
    - *(string) --* 

    

  
  :rtype: dict
  :returns:

              The first line of the address.

              The second line of the address, if applicable.

              The third line of the address, if applicable.

              
            

            - **DistrictOrCounty** *(string) --*

              The city that the address is in.

              The country code for the country the address is in.
