======
Client
======



.. py:class:: PrivateCAConnectorforSCEP.Client

  A low-level client representing Private CA Connector for SCEP
  

  Connector for SCEP creates a connector between Amazon Web Services Private CA and your SCEP-enabled clients and devices. For more information, see `Connector for SCEP <https://docs.aws.amazon.com/privateca/latest/userguide/scep-connector.htmlconnector-for-scep.html>`__ in the *Amazon Web Services Private CA User Guide*.

  ::

    
    import boto3
    
    client = boto3.client('pca-connector-scep')
