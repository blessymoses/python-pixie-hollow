delete_intent_version
*********************



.. py:method:: LexModelBuildingService.Client.delete_intent_version(**kwargs)

  

  Deletes a specific version of an intent. To delete all versions of a intent, use the  DeleteIntent operation.

   

  This operation requires permissions for the ``lex:DeleteIntentVersion`` action.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/lex-models-2017-04-19/DeleteIntentVersion>`_  


  **Request Syntax**
  ::

    response = client.delete_intent_version(
        name='string',
        version='string'
    )
    
  :type name: string
  :param name: **[REQUIRED]** 

    The name of the intent.

    

  
  :type version: string
  :param version: **[REQUIRED]** 

    The version of the intent to delete. You cannot delete the ``$LATEST`` version of the intent. To delete the ``$LATEST`` version, use the  DeleteIntent operation.

    

  
  
  :returns: None
