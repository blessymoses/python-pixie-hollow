********************
create_domain_layout
********************



.. py:method:: CustomerProfiles.Client.create_domain_layout(**kwargs)

  

  Creates the layout to view data for a specific domain. This API can only be invoked from the Amazon Connect admin website.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/customer-profiles-2020-08-15/CreateDomainLayout>`_  


  **Request Syntax**
  ::

    response = client.create_domain_layout(
        DomainName='string',
        LayoutDefinitionName='string',
        Description='string',
        DisplayName='string',
        IsDefault=True|False,
        LayoutType='PROFILE_EXPLORER',
        Layout='string',
        Tags={
            'string': 'string'
        }
    )
    
  :type DomainName: string
  :param DomainName: **[REQUIRED]** 

    The unique name of the domain.

    

  
  :type LayoutDefinitionName: string
  :param LayoutDefinitionName: **[REQUIRED]** 

    The unique name of the layout.

    

  
  :type Description: string
  :param Description: **[REQUIRED]** 

    The description of the layout

    

  
  :type DisplayName: string
  :param DisplayName: **[REQUIRED]** 

    The display name of the layout

    

  
  :type IsDefault: boolean
  :param IsDefault: 

    If set to true for a layout, this layout will be used by default to view data. If set to false, then the layout will not be used by default, but it can be used to view data by explicitly selecting it in the console.

    

  
  :type LayoutType: string
  :param LayoutType: **[REQUIRED]** 

    The type of layout that can be used to view data under a Customer Profiles domain.

    

  
  :type Layout: string
  :param Layout: **[REQUIRED]** 

    A customizable layout that can be used to view data under a Customer Profiles domain.

    

  
  :type Tags: dict
  :param Tags: 

    The tags used to organize, track, or control access for this resource.
