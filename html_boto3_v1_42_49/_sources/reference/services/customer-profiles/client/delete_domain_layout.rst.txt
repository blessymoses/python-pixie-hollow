delete_domain_layout
********************



.. py:method:: CustomerProfiles.Client.delete_domain_layout(**kwargs)

  

  Deletes the layout used to view data for a specific domain. This API can only be invoked from the Amazon Connect admin website.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/customer-profiles-2020-08-15/DeleteDomainLayout>`_  


  **Request Syntax**

    response = client.delete_domain_layout(
        DomainName='string',
        LayoutDefinitionName='string'
    )
    
  :type DomainName: string
  :param DomainName: **[REQUIRED]**

  :type LayoutDefinitionName: string
