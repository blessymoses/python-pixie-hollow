delete_document_version
***********************



.. py:method:: WorkDocs.Client.delete_document_version(**kwargs)

  

  Deletes a specific version of a document.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/workdocs-2016-05-01/DeleteDocumentVersion>`_  


  **Request Syntax**
  ::

    response = client.delete_document_version(
        AuthenticationToken='string',
        DocumentId='string',
        VersionId='string',
        DeletePriorVersions=True|False
    )
    
  :type AuthenticationToken: string
  :param AuthenticationToken: 

    Amazon WorkDocs authentication token. Not required when using Amazon Web Services administrator credentials to access the API.

    

  
  :type DocumentId: string
  :param DocumentId: **[REQUIRED]** 

    The ID of the document associated with the version being deleted.

    

  
  :type VersionId: string
  :param VersionId: **[REQUIRED]** 

    The ID of the version being deleted.

    

  
  :type DeletePriorVersions: boolean
  :param DeletePriorVersions: **[REQUIRED]** 

    Deletes all versions of a document prior to the current version.

    

  
  
  :returns: None
