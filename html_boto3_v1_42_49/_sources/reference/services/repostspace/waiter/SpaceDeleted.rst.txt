  .. py:method:: wait(**kwargs)

    Polls :py:meth:`rePostPrivate.Client.get_space` every 300 seconds until a successful state is reached. An error is raised after 24 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/repostspace-2022-05-13/GetSpace>`_    


    **Request Syntax**
    ::

      waiter.wait(
          spaceId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type spaceId: string
    :param spaceId: **[REQUIRED]** 

      The ID of the private re:Post.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 300

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 24

        

      
    
    
    :returns: None
