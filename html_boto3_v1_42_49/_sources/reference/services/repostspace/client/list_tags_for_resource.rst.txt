.. py:method:: rePostPrivate.Client.list_tags_for_resource(**kwargs)

  

  Returns the tags that are associated with the AWS re:Post Private resource specified by the resourceArn. The only resource that can be tagged is a private re:Post.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/repostspace-2022-05-13/ListTagsForResource>`_

        resourceArn='string'
