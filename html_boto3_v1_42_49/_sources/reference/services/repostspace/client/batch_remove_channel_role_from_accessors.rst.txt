batch_remove_channel_role_from_accessors
****************************************



.. py:method:: rePostPrivate.Client.batch_remove_channel_role_from_accessors(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/repostspace-2022-05-13/BatchRemoveChannelRoleFromAccessors>`_  


  **Request Syntax**

        accessorIds=[

        ],
        channelRole='ASKER'|'EXPERT'|'MODERATOR'|'SUPPORTREQUESTOR'
    )

  :param spaceId: **[REQUIRED]** 

    The unique ID of the private re:Post.

  :param channelId: **[REQUIRED]**

  :type accessorIds: list
  :param accessorIds: **[REQUIRED]**
