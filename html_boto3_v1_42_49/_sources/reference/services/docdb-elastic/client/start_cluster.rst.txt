*************
start_cluster
*************



.. py:method:: DocDBElastic.Client.start_cluster(**kwargs)

  

  Restarts the stopped elastic cluster that is specified by ``clusterARN``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/docdb-elastic-2022-11-28/StartCluster>`_  


  **Request Syntax**
  ::

    response = client.start_cluster(
        clusterArn='string'
    )
    
  :type clusterArn: string
  :param clusterArn: **[REQUIRED]** 

    The ARN identifier of the elastic cluster.

    

  
  
  :rtype: dict
  :returns:
