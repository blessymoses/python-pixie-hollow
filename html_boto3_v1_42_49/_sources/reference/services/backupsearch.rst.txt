BackupSearch

======
Client
======



.. py:class:: BackupSearch.Client

  A low-level client representing AWS Backup Search
  

   

  Backup Search is the recovery point and item level search for Backup.

   

  For additional information, see:

   

  
  * `Backup API Reference <https://docs.aws.amazon.com/aws-backup/latest/devguide/api-reference.html>`__
   
  * `Backup Developer Guide <https://docs.aws.amazon.com/aws-backup/latest/devguide/whatisbackup.html>`__
  

  

  ::

    
    import boto3
    
    client = boto3.client('backupsearch')
