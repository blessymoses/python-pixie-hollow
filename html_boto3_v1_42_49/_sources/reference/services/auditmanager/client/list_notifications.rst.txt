list_notifications

.. py:method:: AuditManager.Client.list_notifications(**kwargs)

  

  Returns a list of all Audit Manager notifications.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/auditmanager-2017-07-25/ListNotifications>`_  


  **Request Syntax**
  ::

    response = client.list_notifications(
        nextToken='string',
        maxResults=123
    )
    
  :type nextToken: string
  :param nextToken: 

    The pagination token that's used to fetch the next set of results.

    

  
  :type maxResults: integer
  :param maxResults: 

    Represents the maximum number of results on a page or for an API request call.

    

  
  
  :rtype: dict
  :returns:
