register_account

.. py:method:: AuditManager.Client.register_account(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/auditmanager-2017-07-25/RegisterAccount>`_  


  **Request Syntax**

        kmsKey='string',
        delegatedAdminAccount='string'
    )
    
  :type kmsKey: string
  :param kmsKey: 

    The KMS key details.

    

  
  :type delegatedAdminAccount: string
  :param delegatedAdminAccount:

  :returns:
