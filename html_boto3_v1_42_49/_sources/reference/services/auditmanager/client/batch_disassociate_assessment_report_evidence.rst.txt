.. py:method:: AuditManager.Client.batch_disassociate_assessment_report_evidence(**kwargs)

  **Request Syntax**

        assessmentId='string',
        evidenceFolderId='string',
        evidenceIds=[
            'string',
        ]
    )

  :param assessmentId: **[REQUIRED]**
