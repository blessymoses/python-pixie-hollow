  .. py:method:: wait(**kwargs)

    Polls :py:meth:`CodeGuruReviewer.Client.describe_code_review` every 10 seconds until a successful state is reached. An error is raised after 180 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/codeguru-reviewer-2019-09-19/DescribeCodeReview>`_    


    **Request Syntax**
    ::

      waiter.wait(
          CodeReviewArn='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type CodeReviewArn: string
    :param CodeReviewArn: **[REQUIRED]** 

      The Amazon Resource Name (ARN) of the `CodeReview <https://docs.aws.amazon.com/codeguru/latest/reviewer-api/API_CodeReview.html>`__ object.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 10

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 180

        

      
    
    
    :returns: None
