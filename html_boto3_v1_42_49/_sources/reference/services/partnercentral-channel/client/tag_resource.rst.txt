.. py:method:: PartnerCentralChannelAPI.Client.tag_resource(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/partnercentral-channel-2024-03-18/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        resourceArn='string',
        tags=[
            {

                'value': 'string'
            },
        ]

  :type resourceArn: string
  :param resourceArn: **[REQUIRED]**
