tag_resource
************



.. py:method:: Detective.Client.tag_resource(**kwargs)

  

  Applies tag values to a behavior graph.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/detective-2018-10-26/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        ResourceArn='string',
        Tags={
            'string': 'string'
        }
    )
    
  :type ResourceArn: string
  :param ResourceArn: **[REQUIRED]** 

    The ARN of the behavior graph to assign the tags to.

    

  
  :type Tags: dict
  :param Tags: **[REQUIRED]** 

    The tags to assign to the behavior graph. You can add up to 50 tags. For each tag, you provide the tag key and the tag value. Each tag key can contain up to 128 characters. Each tag value can contain up to 256 characters.
