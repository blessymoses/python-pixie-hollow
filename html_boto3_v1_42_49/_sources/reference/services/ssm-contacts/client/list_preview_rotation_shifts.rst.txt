list_preview_rotation_shifts
****************************



.. py:method:: SSMContacts.Client.list_preview_rotation_shifts(**kwargs)

  

  Returns a list of shifts based on rotation configuration parameters.

   

  .. note::

    

    The Incident Manager primarily uses this operation to populate the **Preview** calendar. It is not typically run by end users.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ssm-contacts-2021-05-03/ListPreviewRotationShifts>`_
