infer_snomedct
**************



.. py:method:: ComprehendMedical.Client.infer_snomedct(**kwargs)

  

  InferSNOMEDCT detects possible medical concepts as entities and links them to codes from the Systematized Nomenclature of Medicine, Clinical Terms (SNOMED-CT) ontology

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/comprehendmedical-2018-10-30/InferSNOMEDCT>`_  


  **Request Syntax**
  ::

    response = client.infer_snomedct(
        Text='string'
    )
    
  :type Text: string
  :param Text: **[REQUIRED]** 

    The input text to be analyzed using InferSNOMEDCT.

    

  
  
  :rtype: dict
  :returns:
