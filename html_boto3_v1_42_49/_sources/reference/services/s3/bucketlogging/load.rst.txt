load

.. py:method:: S3.BucketLogging.load()

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/s3-2006-03-01/None>`_  


  **Request Syntax**

  ::

    bucket_logging.load()
  :returns: None
