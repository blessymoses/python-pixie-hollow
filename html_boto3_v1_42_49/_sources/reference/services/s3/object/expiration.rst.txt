**********
expiration
**********



.. py:attribute:: S3.Object.expiration

  

  - *(string) --* 

    If the object expiration is configured (see `PutBucketLifecycleConfiguration <https://docs.aws.amazon.com/AmazonS3/latest/API/API_PutBucketLifecycleConfiguration.html>`__), the response includes this header. It includes the ``expiry-date`` and ``rule-id`` key-value pairs providing object expiration information. The value of the ``rule-id`` is URL-encoded.

     

    .. note::

      

      Object expiration information is not returned in directory buckets and this header returns the value " ``NotImplemented``" in all responses for directory buckets.
