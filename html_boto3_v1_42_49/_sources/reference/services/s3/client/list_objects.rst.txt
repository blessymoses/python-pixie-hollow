************
list_objects
************



.. py:method:: S3.Client.list_objects(**kwargs)

  

  .. note::

    

    This operation is not supported for directory buckets.

    

   

  Returns some or all (up to 1,000) of the objects in a bucket. You can use the request parameters as selection criteria to return a subset of the objects in a bucket. A 200 OK response can contain valid or invalid XML. Be sure to design your application to parse the contents of the response and handle it appropriately.

    Encoding type used by Amazon S3 to encode the `object keys <https://docs.aws.amazon.com/AmazonS3/latest/userguide/object-keys.html>`__ in the response. Responses are encoded only in UTF-8.     .. note::
