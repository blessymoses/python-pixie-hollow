======
Client
======



.. py:class:: BedrockRuntime.Client

  A low-level client representing Amazon Bedrock Runtime
  

  Describes the API operations for running inference using Amazon Bedrock models.

  ::

    
    import boto3
    
    client = boto3.client('bedrock-runtime')
