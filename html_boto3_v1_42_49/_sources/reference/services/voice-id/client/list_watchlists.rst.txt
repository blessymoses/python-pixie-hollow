list_watchlists

.. py:method:: VoiceID.Client.list_watchlists(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/voice-id-2021-09-27/ListWatchlists>`_

        DomainId='string',
        MaxResults=123,
        NextToken='string'
    )

  :param DomainId: **[REQUIRED]** 

    The identifier of the domain.

    

  
  :type MaxResults: integer
  :param MaxResults:
