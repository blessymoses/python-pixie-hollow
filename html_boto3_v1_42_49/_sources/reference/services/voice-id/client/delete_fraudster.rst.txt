****************
delete_fraudster
****************



.. py:method:: VoiceID.Client.delete_fraudster(**kwargs)

  

  Deletes the specified fraudster from Voice ID. This action disassociates the fraudster from any watchlists it is a part of.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/voice-id-2021-09-27/DeleteFraudster>`_  


  **Request Syntax**
  ::

    response = client.delete_fraudster(
        DomainId='string',
        FraudsterId='string'
    )
    
  :type DomainId: string
  :param DomainId: **[REQUIRED]** 

    The identifier of the domain that contains the fraudster.

    

  
  :type FraudsterId: string
  :param FraudsterId: **[REQUIRED]** 

    The identifier of the fraudster you want to delete.

    

  
  
  :returns: None
