.. py:method:: QConnect.Client.get_quick_response(**kwargs)

  

  Retrieves the quick response.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/qconnect-2020-10-19/GetQuickResponse>`_  


  **Request Syntax**
  ::

    response = client.get_quick_response(
        quickResponseId='string',
        knowledgeBaseId='string'
    )
    
  :type quickResponseId: string
  :param quickResponseId: **[REQUIRED]** 

    The identifier of the quick response.

    

  
  :type knowledgeBaseId: string
  :param knowledgeBaseId: **[REQUIRED]** 

    The identifier of the knowledge base. This should be a QUICK_RESPONSES type knowledge base.

    

  
  
  :rtype: dict
  :returns:
