****************
update_blueprint
****************



.. py:method:: Glue.Client.update_blueprint(**kwargs)

  

  Updates a registered blueprint.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/glue-2017-03-31/UpdateBlueprint>`_  


  **Request Syntax**
  ::

    response = client.update_blueprint(
        Name='string',
        Description='string',
        BlueprintLocation='string'
    )
    
  :type Name: string
  :param Name: **[REQUIRED]** 

    The name of the blueprint.

    

  
  :type Description: string
  :param Description: 

    A description of the blueprint.

    

  
  :type BlueprintLocation: string
  :param BlueprintLocation: **[REQUIRED]** 

    Specifies a path in Amazon S3 where the blueprint is published.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'Name': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **Name** *(string) --* 

        Returns the name of the blueprint that was updated.
