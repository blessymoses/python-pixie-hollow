get_usage_profile

.. py:method:: Glue.Client.get_usage_profile(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/glue-2017-03-31/GetUsageProfile>`_

    response = client.get_usage_profile(
        Name='string'
