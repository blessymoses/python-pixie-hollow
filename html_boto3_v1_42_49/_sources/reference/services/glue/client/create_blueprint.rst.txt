.. py:method:: Glue.Client.create_blueprint(**kwargs)

  **Request Syntax**
  ::

    response = client.create_blueprint(
        Name='string',
        Description='string',
        BlueprintLocation='string',
        Tags={
            'string': 'string'
        }
    )

  :param Name: **[REQUIRED]**
