delete_invitations
******************



.. py:method:: Macie2.Client.delete_invitations(**kwargs)

  

  Deletes Amazon Macie membership invitations that were received from specific accounts.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/macie2-2020-01-01/DeleteInvitations>`_  


  **Request Syntax**
  ::

    response = client.delete_invitations(
        accountIds=[
            'string',
        ]
    )
    
  :type accountIds: list
  :param accountIds: **[REQUIRED]** 

    An array that lists Amazon Web Services account IDs, one for each account that sent an invitation to delete.

    

  
    - *(string) --* 

    

  
  :rtype: dict
  :returns:

    ::

      {
          'unprocessedAccounts': [
              {
                  'accountId': 'string',
                  'errorCode': 'ClientError'|'InternalError',
                  'errorMessage': 'string'
              },
          ]
      }
