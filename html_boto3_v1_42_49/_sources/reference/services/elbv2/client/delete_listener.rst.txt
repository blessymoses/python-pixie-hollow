delete_listener
***************



.. py:method:: ElasticLoadBalancingv2.Client.delete_listener(**kwargs)

  

  Deletes the specified listener.

   

  Alternatively, your listener is deleted when you delete the load balancer to which it is attached.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/elasticloadbalancingv2-2015-12-01/DeleteListener>`_  


  **Request Syntax**
  ::

    response = client.delete_listener(
        ListenerArn='string'
    )
    
  :type ListenerArn: string
  :param ListenerArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the listener.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {}
      
    **Response Structure**

    

    - *(dict) --* 
  
  **Exceptions**
  
  *   :py:class:`ElasticLoadBalancingv2.Client.exceptions.ListenerNotFoundException`

  
  *   :py:class:`ElasticLoadBalancingv2.Client.exceptions.ResourceInUseException`

  

  **Examples**

  This example deletes the specified listener.
  ::

    response = client.delete_listener(
        ListenerArn='arn:aws:elasticloadbalancing:ua-west-2:123456789012:listener/app/my-load-balancer/50dc6c495c0c9188/f2f7dc8efc522ab2',
    )
    
    print(response)
