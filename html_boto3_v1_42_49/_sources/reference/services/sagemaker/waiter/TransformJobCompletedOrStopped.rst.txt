  .. py:method:: wait(**kwargs)

    Polls :py:meth:`SageMaker.Client.describe_transform_job` every 60 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sagemaker-2017-07-24/DescribeTransformJob>`_    


    **Request Syntax**
    ::

      waiter.wait(
          TransformJobName='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type TransformJobName: string
    :param TransformJobName: **[REQUIRED]** 

      The name of the transform job that you want to view details of.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 60

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60

        

      
    
    
    :returns: None
