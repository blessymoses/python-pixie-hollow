  .. py:method:: wait(**kwargs)

    Polls :py:meth:`SageMaker.Client.describe_processing_job` every 60 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sagemaker-2017-07-24/DescribeProcessingJob>`_    


    **Request Syntax**
    ::

      waiter.wait(
          ProcessingJobName='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type ProcessingJobName: string
    :param ProcessingJobName: **[REQUIRED]** 

      The name of the processing job. The name must be unique within an Amazon Web Services Region in the Amazon Web Services account.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 60

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60

        

      
    
    
    :returns: None
