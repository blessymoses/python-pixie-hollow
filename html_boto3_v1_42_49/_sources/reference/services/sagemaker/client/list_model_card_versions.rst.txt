list_model_card_versions
************************



.. py:method:: SageMaker.Client.list_model_card_versions(**kwargs)

  

  List existing versions of an Amazon SageMaker Model Card.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sagemaker-2017-07-24/ListModelCardVersions>`_  


  **Request Syntax**
  ::

    response = client.list_model_card_versions(
        CreationTimeAfter=datetime(2015, 1, 1),
        CreationTimeBefore=datetime(2015, 1, 1),
        MaxResults=123,
        ModelCardName='string',
        ModelCardStatus='Draft'|'PendingReview'|'Approved'|'Archived',
        NextToken='string',
        SortBy='Version',
        SortOrder='Ascending'|'Descending'
    )
    
  :type CreationTimeAfter: datetime
  :param CreationTimeAfter: 

    Only list model card versions that were created after the time specified.

    

  
  :type CreationTimeBefore: datetime
  :param CreationTimeBefore: 

    Only list model card versions that were created before the time specified.

    

  
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of model card versions to list.

    

  
  :type ModelCardName: string
  :param ModelCardName: **[REQUIRED]** 

    List model card versions for the model card with the specified name or Amazon Resource Name (ARN).

    

  
  :type ModelCardStatus: string
  :param ModelCardStatus: 

    Only list model card versions with the specified approval status.

    

  
  :type NextToken: string
  :param NextToken: 

    If the response to a previous ``ListModelCardVersions`` request was truncated, the response includes a ``NextToken``. To retrieve the next set of model card versions, use the token in the next request.

    

  
  :type SortBy: string
  :param SortBy: 

    Sort listed model card versions by version. Sorts by version by default.

    

  
  :type SortOrder: string
  :param SortOrder: 

    Sort model card versions by ascending or descending order.

    

  
  
  :rtype: dict
  :returns:
