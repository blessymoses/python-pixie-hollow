***************
create_artifact
***************



.. py:method:: SageMaker.Client.create_artifact(**kwargs)

  

  Creates an *artifact*. An artifact is a lineage tracking entity that represents a URI addressable object or data. Some examples are the S3 URI of a dataset and the ECR registry path of an image. For more information, see `Amazon SageMaker ML Lineage Tracking <https://docs.aws.amazon.com/sagemaker/latest/dg/lineage-tracking.html>`__.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sagemaker-2017-07-24/CreateArtifact>`_  


  **Request Syntax**
  ::

    response = client.create_artifact(
        ArtifactName='string',
        Source={
            'SourceUri': 'string',
            'SourceTypes': [
                {
                    'SourceIdType': 'MD5Hash'|'S3ETag'|'S3Version'|'Custom',
                    'Value': 'string'
                },
            ]
        },
        ArtifactType='string',
        Properties={
            'string': 'string'
        },
        MetadataProperties={
            'CommitId': 'string',
            'Repository': 'string',
            'GeneratedBy': 'string',
            'ProjectId': 'string'
        },
        Tags=[
            {
                'Key': 'string',
                'Value': 'string'
            },
        ]
    )
