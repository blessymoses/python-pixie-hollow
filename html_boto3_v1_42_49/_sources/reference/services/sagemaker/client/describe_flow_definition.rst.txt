describe_flow_definition
************************



.. py:method:: SageMaker.Client.describe_flow_definition(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sagemaker-2017-07-24/DescribeFlowDefinition>`_
