create_human_task_ui
********************



.. py:method:: SageMaker.Client.create_human_task_ui(**kwargs)

  

  Defines the settings you will use for the human review workflow user interface. Reviewers will see a three-panel interface with an instruction area, the item to review, and an input area.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sagemaker-2017-07-24/CreateHumanTaskUi>`_  


  **Request Syntax**
  ::

    response = client.create_human_task_ui(
        HumanTaskUiName='string',
        UiTemplate={
            'Content': 'string'
        },
        Tags=[
            {
                'Key': 'string',
                'Value': 'string'
            },
        ]
    )
    
  :type HumanTaskUiName: string
  :param HumanTaskUiName: **[REQUIRED]** 

    The name of the user interface you are creating.

    

  
  :type UiTemplate: dict
  :param UiTemplate: **[REQUIRED]** 

    The Liquid template for the worker user interface.

    

  
    - **Content** *(string) --* **[REQUIRED]** 

      The content of the Liquid template for the worker user interface.

      

    
  
  :type Tags: list
  :param Tags:
