list_tags_for_domain
********************



.. py:method:: Route53Domains.Client.list_tags_for_domain(**kwargs)

  

  This operation returns all of the tags that are associated with the specified domain.

   

  All tag operations are eventually consistent; subsequent operations might not immediately represent all issued operations.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/route53domains-2014-05-15/ListTagsForDomain>`_  


  **Request Syntax**
  ::

    response = client.list_tags_for_domain(
        DomainName='string'
    )
    
  :type DomainName: string
  :param DomainName: **[REQUIRED]** 

    The domain for which you want to get a list of tags.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'TagList': [
              {
                  'Key': 'string',
                  'Value': 'string'
              },
          ]
      }
      
    **Response Structure**

    

    - *(dict) --* 

      The ListTagsForDomain response includes the following elements.

      
      

      - **TagList** *(list) --* 

        A list of the tags that are associated with the specified domain.

        
        

        - *(dict) --* 

          Each tag includes the following elements.

          
          

          - **Key** *(string) --* 

            The key (name) of a tag.

             

            Valid values: A-Z, a-z, 0-9, space, ".:/=+\-@"

             

            Constraints: Each key can be 1-128 characters long.

            
          

          - **Value** *(string) --* 

            The value of a tag.

             

            Valid values: A-Z, a-z, 0-9, space, ".:/=+\-@"

             

            Constraints: Each value can be 0-256 characters long.
