.. py:class:: ConnectCampaignServiceV2.Client

  A low-level client representing AmazonConnectCampaignServiceV2
  

  Provide APIs to create and manage Amazon Connect Campaigns.

  ::

    
    import boto3
    
    client = boto3.client('connectcampaignsv2')
