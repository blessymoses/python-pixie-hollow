***************
list_db_servers
***************



.. py:method:: odb.Client.list_db_servers(**kwargs)

  

  Returns information about the database servers that belong to the specified Exadata infrastructure.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/odb-2024-08-20/ListDbServers>`_  


  **Request Syntax**
  ::

    response = client.list_db_servers(
        cloudExadataInfrastructureId='string',
        maxResults=123,
        nextToken='string'
    )
    
  :type cloudExadataInfrastructureId: string
  :param cloudExadataInfrastructureId: **[REQUIRED]** 

    The unique identifier of the Oracle Exadata infrastructure.

    

  
  :type maxResults: integer
  :param maxResults: 

    The maximum number of items to return for this request. To get the next page of items, make another request with the token returned in the output.

     

    Default: ``10``

    

  
  :type nextToken: string
  :param nextToken: 

    The token returned from a previous paginated request. Pagination continues from the end of the items returned by the previous request.

    

  
  
  :rtype: dict
  :returns:
