  .. py:method:: wait(**kwargs)

    Polls :py:meth:`MediaLive.Client.describe_input` every 5 seconds until a successful state is reached. An error is raised after 84 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/medialive-2017-10-14/DescribeInput>`_    


    **Request Syntax**
    ::

      waiter.wait(
          InputId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type InputId: string
    :param InputId: **[REQUIRED]** Unique ID of the input

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 5

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 84

        

      
    
    
    :returns: None
