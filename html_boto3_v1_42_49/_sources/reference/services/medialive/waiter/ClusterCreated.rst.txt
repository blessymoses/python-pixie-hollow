  .. py:method:: wait(**kwargs)

    Polls :py:meth:`MediaLive.Client.describe_cluster` every 3 seconds until a successful state is reached. An error is raised after 5 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/medialive-2017-10-14/DescribeCluster>`_    


    **Request Syntax**
    ::

      waiter.wait(
          ClusterId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type ClusterId: string
    :param ClusterId: **[REQUIRED]** The ID of the cluster.

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 3

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 5

        

      
    
    
    :returns: None
