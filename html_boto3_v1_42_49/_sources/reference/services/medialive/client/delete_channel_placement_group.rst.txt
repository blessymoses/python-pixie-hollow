delete_channel_placement_group

.. py:method:: MediaLive.Client.delete_channel_placement_group(**kwargs)

        ChannelPlacementGroupId='string',
        ClusterId='string'

  :param ChannelPlacementGroupId: **[REQUIRED]** The ID of the channel placement group.

  
  :type ClusterId: string
  :param ClusterId: **[REQUIRED]** The ID of the cluster.

  
  
  :rtype: dict
  :returns:
