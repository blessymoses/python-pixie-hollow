*****************
stop_input_device
*****************



.. py:method:: MediaLive.Client.stop_input_device(**kwargs)

  Stop an input device that is attached to a MediaConnect flow. (There is no need to stop a device that is attached to a MediaLive input; MediaLive automatically stops the device when the channel stops.)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/medialive-2017-10-14/StopInputDevice>`_  


  **Request Syntax**
  ::

    response = client.stop_input_device(
        InputDeviceId='string'
    )
    
  :type InputDeviceId: string
  :param InputDeviceId: **[REQUIRED]** The unique ID of the input device to stop. For example, hd-123456789abcdef.

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {}
      
    **Response Structure**

    

    - *(dict) --* Your stop request has succeeded.
