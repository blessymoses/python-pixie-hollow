======
Client
======



.. py:class:: S3Control.Client

  A low-level client representing AWS S3 Control
  

  Amazon Web Services S3 Control provides access to Amazon S3 control plane actions.

  ::

    
    import boto3
    
    client = boto3.client('s3control')
