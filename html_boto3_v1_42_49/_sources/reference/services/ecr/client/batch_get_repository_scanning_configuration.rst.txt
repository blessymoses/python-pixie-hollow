.. py:method:: ECR.Client.batch_get_repository_scanning_configuration(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ecr-2015-09-21/BatchGetRepositoryScanningConfiguration>`_
