  .. py:method:: wait(**kwargs)

    Polls :py:meth:`DeadlineCloud.Client.get_queue_limit_association` every 10 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/deadline-2023-10-12/GetQueueLimitAssociation>`_    


    **Request Syntax**
    ::

      waiter.wait(
          farmId='string',
          queueId='string',
          limitId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type farmId: string
    :param farmId: **[REQUIRED]** 

      The unique identifier of the farm that contains the associated queue and limit.

      

    
    :type queueId: string
    :param queueId: **[REQUIRED]** 

      The unique identifier of the queue associated with the limit.

      

    
    :type limitId: string
    :param limitId: **[REQUIRED]** 

      The unique identifier of the limit associated with the queue.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 10

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60
