**********************
list_sensor_statistics
**********************



.. py:method:: LookoutEquipment.Client.list_sensor_statistics(**kwargs)

  

  Lists statistics about the data collected for each of the sensors that have been successfully ingested in the particular dataset. Can also be used to retreive Sensor Statistics for a previous ingestion job.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/lookoutequipment-2020-12-15/ListSensorStatistics>`_  


  **Request Syntax**
  ::

    response = client.list_sensor_statistics(
        DatasetName='string',
        IngestionJobId='string',
        MaxResults=123,
        NextToken='string'
    )
    
  :type DatasetName: string
  :param DatasetName: **[REQUIRED]** 

    The name of the dataset associated with the list of Sensor Statistics.

    

  
  :type IngestionJobId: string
  :param IngestionJobId: 

    The ingestion job id associated with the list of Sensor Statistics. To get sensor statistics for a particular ingestion job id, both dataset name and ingestion job id must be submitted as inputs.

    

  
  :type MaxResults: integer
  :param MaxResults: 

    Specifies the maximum number of sensors for which to retrieve statistics.

    

  
  :type NextToken: string
  :param NextToken: 

    An opaque pagination token indicating where to continue the listing of sensor statistics.

    

  
  
  :rtype: dict
  :returns:
