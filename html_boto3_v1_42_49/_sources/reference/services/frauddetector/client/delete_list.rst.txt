delete_list
***********



.. py:method:: FraudDetector.Client.delete_list(**kwargs)

  

  Deletes the list, provided it is not used in a rule.

   

  When you delete a list, Amazon Fraud Detector permanently deletes that list and the elements in the list.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/frauddetector-2019-11-15/DeleteList>`_  


  **Request Syntax**
  ::

    response = client.delete_list(
        name='string'
    )
    
  :type name: string
  :param name: **[REQUIRED]** 

    The name of the list to delete.
