deregister_db_proxy_targets
***************************



.. py:method:: RDS.Client.deregister_db_proxy_targets(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rds-2014-10-31/DeregisterDBProxyTargets>`_

            'string',
        ]
    )
