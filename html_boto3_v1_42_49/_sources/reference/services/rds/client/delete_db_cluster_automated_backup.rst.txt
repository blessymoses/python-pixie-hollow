**********************************
delete_db_cluster_automated_backup
**********************************



.. py:method:: RDS.Client.delete_db_cluster_automated_backup(**kwargs)

  

  Deletes automated backups using the ``DbClusterResourceId`` value of the source DB cluster or the Amazon Resource Name (ARN) of the automated backups.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rds-2014-10-31/DeleteDBClusterAutomatedBackup>`_  


  **Request Syntax**
  ::

    response = client.delete_db_cluster_automated_backup(
        DbClusterResourceId='string'
    )
    
  :type DbClusterResourceId: string
  :param DbClusterResourceId: **[REQUIRED]** 

    The identifier for the source DB cluster, which can't be changed and which is unique to an Amazon Web Services Region.

    

  
  
  :rtype: dict
  :returns:
