*****************
put_account_alias
*****************



.. py:method:: SupportApp.Client.put_account_alias(**kwargs)

  

  Creates or updates an individual alias for each Amazon Web Services account ID. The alias appears in the Amazon Web Services Support App page of the Amazon Web Services Support Center. The alias also appears in Slack messages from the Amazon Web Services Support App.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/support-app-2021-08-20/PutAccountAlias>`_  


  **Request Syntax**
  ::

    response = client.put_account_alias(
        accountAlias='string'
    )
    
  :type accountAlias: string
  :param accountAlias: **[REQUIRED]** 

    An alias or short name for an Amazon Web Services account.
