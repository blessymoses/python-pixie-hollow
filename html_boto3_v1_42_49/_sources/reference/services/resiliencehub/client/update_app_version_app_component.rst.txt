update_app_version_app_component
********************************



.. py:method:: ResilienceHub.Client.update_app_version_app_component(**kwargs)

  

  Updates an existing Application Component in the Resilience Hub application.

   

  .. note::

    

    This API updates the Resilience Hub application draft version. To use this Application Component for running assessments, you must publish the Resilience Hub application using the ``PublishAppVersion`` API.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/resiliencehub-2020-04-30/UpdateAppVersionAppComponent>`_

        type='string'
