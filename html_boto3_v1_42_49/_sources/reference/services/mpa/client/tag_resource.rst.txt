tag_resource
************



.. py:method:: MultipartyApproval.Client.tag_resource(**kwargs)

  

  Creates or updates a resource tag. Each tag is a label consisting of a user-defined key and value. Tags can help you manage, identify, organize, search for, and filter resources.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mpa-2022-07-26/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        ResourceArn='string',
        Tags={
            'string': 'string'
        }
    )
    
  :type ResourceArn: string
  :param ResourceArn: **[REQUIRED]** 

    Amazon Resource Name (ARN) for the resource you want to tag.

    

  
  :type Tags: dict
  :param Tags: **[REQUIRED]** 

    Tags that you have added to the specified resource.
