start_domain_verification
*************************



.. py:method:: VPCLattice.Client.start_domain_verification(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/vpc-lattice-2022-11-30/StartDomainVerification>`_  


  **Request Syntax**

    response = client.start_domain_verification(
        clientToken='string',
        domainName='string',
        tags={
            'string': 'string'
        }
    )

  :param clientToken: 

    A unique, case-sensitive identifier that you provide to ensure the idempotency of the request. If you retry a request that completed successfully using the same client token and parameters, the retry succeeds without performing any actions. If the parameters aren't identical, the retry fails.

  :type domainName: string
