***********
remove_tags
***********



.. py:method:: EMR.Client.remove_tags(**kwargs)

  

  Removes tags from an Amazon EMR resource, such as a cluster or Amazon EMR Studio. Tags make it easier to associate resources in various ways, such as grouping clusters to track your Amazon EMR resource allocation costs. For more information, see `Tag Clusters <https://docs.aws.amazon.com/emr/latest/ManagementGuide/emr-plan-tags.html>`__.

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/elasticmapreduce-2009-03-31/RemoveTags>`_  


  **Request Syntax**
  ::

    response = client.remove_tags(
        ResourceId='string',
        TagKeys=[
            'string',
        ]
    )
    
  :type ResourceId: string
  :param ResourceId: **[REQUIRED]** 

    The Amazon EMR resource identifier from which tags will be removed. For example, a cluster identifier or an Amazon EMR Studio ID.

    

  
  :type TagKeys: list
  :param TagKeys: **[REQUIRED]** 

    A list of tag keys to remove from the resource.
