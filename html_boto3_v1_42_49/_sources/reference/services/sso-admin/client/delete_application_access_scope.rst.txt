delete_application_access_scope
*******************************



.. py:method:: SSOAdmin.Client.delete_application_access_scope(**kwargs)

  

  Deletes an IAM Identity Center access scope from an application.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sso-admin-2020-07-20/DeleteApplicationAccessScope>`_  


  **Request Syntax**
  ::

    response = client.delete_application_access_scope(
        ApplicationArn='string',
        Scope='string'
    )
    
  :type ApplicationArn: string
  :param ApplicationArn: **[REQUIRED]** 

    Specifies the ARN of the application with the access scope to delete.

    

  
  :type Scope: string
  :param Scope: **[REQUIRED]** 

    Specifies the name of the access scope to remove from the application.

    

  
  
  :returns: None
