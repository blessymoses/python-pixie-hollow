***************
delete_instance
***************



.. py:method:: SSOAdmin.Client.delete_instance(**kwargs)

  

  Deletes the instance of IAM Identity Center. Only the account that owns the instance can call this API. Neither the delegated administrator nor member account can delete the organization instance, but those roles can delete their own instance.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sso-admin-2020-07-20/DeleteInstance>`_  


  **Request Syntax**
  ::

    response = client.delete_instance(
        InstanceArn='string'
    )
    
  :type InstanceArn: string
  :param InstanceArn: **[REQUIRED]** 

    The ARN of the instance of IAM Identity Center under which the operation will run.
