******************
get_disk_snapshots
******************



.. py:method:: Lightsail.Client.get_disk_snapshots(**kwargs)

  

  Returns information about all block storage disk snapshots in your AWS account and region.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/lightsail-2016-11-28/GetDiskSnapshots>`_  


  **Request Syntax**
  ::

    response = client.get_disk_snapshots(
        pageToken='string'
    )
    
  :type pageToken: string
  :param pageToken: 

    The token to advance to the next page of results from your request.

     

    To get a page token, perform an initial ``GetDiskSnapshots`` request. If your results are paginated, the response will return a next page token that you can specify as the page token in a subsequent request.

    

  
  
  :rtype: dict
  :returns:
