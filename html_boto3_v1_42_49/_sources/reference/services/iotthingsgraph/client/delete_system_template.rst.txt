delete_system_template
**********************



.. py:method:: IoTThingsGraph.Client.delete_system_template(**kwargs)

  

  Deletes a system. New deployments can't contain the system after its deletion. Existing deployments that contain the system will continue to work because they use a snapshot of the system that is taken when it is deployed.

  

  .. danger::

        This operation is deprecated and may not function as expected. This operation should not be used going forward and is only kept for the purpose of backwards compatiblity.


  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iotthingsgraph-2018-09-06/DeleteSystemTemplate>`_  


  **Request Syntax**
  ::

    response = client.delete_system_template(
        id='string'
    )
    
  :type id: string
  :param id: **[REQUIRED]** 

    The ID of the system to be deleted.

     

    The ID should be in the following format.

     

    ``urn:tdm:REGION/ACCOUNT ID/default:system:SYSTEMNAME``

  :returns:
