***************
update_dns_view
***************



.. py:method:: Route53GlobalResolver.Client.update_dns_view(**kwargs)

  

  Updates the configuration of a DNS view.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/route53globalresolver-2022-09-27/UpdateDNSView>`_  


  **Request Syntax**
  ::

    response = client.update_dns_view(
        dnsViewId='string',
        name='string',
        description='string',
        dnssecValidation='ENABLED'|'DISABLED',
        ednsClientSubnet='ENABLED'|'DISABLED',
        firewallRulesFailOpen='ENABLED'|'DISABLED'
    )
    
  :type dnsViewId: string
  :param dnsViewId: **[REQUIRED]** 

    The unique identifier of the DNS view to update.

    

  
  :type name: string
  :param name: 

    The name of the DNS view.

    

  
  :type description: string
  :param description: 

    A description of the DNS view.

    

  
  :type dnssecValidation: string
  :param dnssecValidation: 

    Whether to enable DNSSEC validation for the DNS view.

    

  
  :type ednsClientSubnet: string
  :param ednsClientSubnet: 

    Whether to enable EDNS Client Subnet injection for the DNS view.

    

  
  :type firewallRulesFailOpen: string
  :param firewallRulesFailOpen: 

    Whether firewall rules should fail open when they cannot be evaluated.

    

  
  
  :rtype: dict
  :returns:
