.. py:method:: Route53GlobalResolver.Client.update_hosted_zone_association(**kwargs)

  

  Updates the configuration of a hosted zone association.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/route53globalresolver-2022-09-27/UpdateHostedZoneAssociation>`_  


  **Request Syntax**
  ::

    response = client.update_hosted_zone_association(
        hostedZoneAssociationId='string',
        name='string'
    )
    
  :type hostedZoneAssociationId: string
  :param hostedZoneAssociationId: **[REQUIRED]** 

    The ID of the private hosted zone association.

    

  
  :type name: string
  :param name: 

    The name you want to update the hosted zone association to.

    

  
  
  :rtype: dict
  :returns:
