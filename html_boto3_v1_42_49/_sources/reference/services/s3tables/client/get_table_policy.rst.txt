****************
get_table_policy
****************



.. py:method:: S3Tables.Client.get_table_policy(**kwargs)

  

  Gets details about a table policy. For more information, see `Viewing a table policy <https://docs.aws.amazon.com/AmazonS3/latest/userguide/s3-tables-table-policy.html#table-policy-get>`__ in the *Amazon Simple Storage Service User Guide*.

    Permissions  

  You must have the ``s3tables:GetTablePolicy`` permission to use this operation.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/s3tables-2018-05-10/GetTablePolicy>`_  


  **Request Syntax**
  ::

    response = client.get_table_policy(
        tableBucketARN='string',
        namespace='string',
        name='string'
    )
    
  :type tableBucketARN: string
  :param tableBucketARN: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the table bucket that contains the table.

    

  
  :type namespace: string
  :param namespace: **[REQUIRED]** 

    The namespace associated with the table.

    

  
  :type name: string
  :param name: **[REQUIRED]** 

    The name of the table.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'resourcePolicy': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **resourcePolicy** *(string) --* 

        The ``JSON`` that defines the policy.
