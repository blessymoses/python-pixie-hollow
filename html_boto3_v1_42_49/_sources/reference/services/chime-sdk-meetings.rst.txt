======
Client
======



.. py:class:: ChimeSDKMeetings.Client

  A low-level client representing Amazon Chime SDK Meetings
  

  The Amazon Chime SDK meetings APIs in this section allow software developers to create Amazon Chime SDK meetings, set the Amazon Web Services Regions for meetings, create and manage users, and send and receive meeting notifications. For more information about the meeting APIs, see `Amazon Chime SDK meetings <https://docs.aws.amazon.com/chime-sdk/latest/APIReference/API_Operations_Amazon_Chime_SDK_Meetings.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('chime-sdk-meetings')
