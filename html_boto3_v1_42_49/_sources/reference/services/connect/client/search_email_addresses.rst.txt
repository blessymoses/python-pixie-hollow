search_email_addresses
**********************



.. py:method:: Connect.Client.search_email_addresses(**kwargs)

  

  Searches email address in an instance, with optional filtering.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/connect-2017-08-08/SearchEmailAddresses>`_  


  **Request Syntax**
  ::

    response = client.search_email_addresses(
        InstanceId='string',
        MaxResults=123,
        NextToken='string',
        SearchCriteria={
            'OrConditions': [
                {'... recursive ...'},
            ],
            'AndConditions': [
                {'... recursive ...'},
            ],
            'StringCondition': {
                'FieldName': 'string',
                'Value': 'string',
                'ComparisonType': 'STARTS_WITH'|'CONTAINS'|'EXACT'
            }
        },
        SearchFilter={
            'TagFilter': {
                'OrConditions': [
                    [
                        {
                            'TagKey': 'string',
                            'TagValue': 'string'
                        },
                    ],
                ],
                'AndConditions': [
                    {
                        'TagKey': 'string',
                        'TagValue': 'string'
                    },
                ],
                'TagCondition': {
                    'TagKey': 'string',
                    'TagValue': 'string'
                }
            }
        }
    )
