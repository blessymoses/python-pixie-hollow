.. py:method:: Connect.Client.list_tags_for_resource(**kwargs)

  

  Lists the tags for the specified resource.

   

  For sample policies that use tags, see `Amazon Connect Identity-Based Policy Examples <https://docs.aws.amazon.com/connect/latest/adminguide/security_iam_id-based-policy-examples.html>`__ in the *Amazon Connect Administrator Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/connect-2017-08-08/ListTagsForResource>`_  


  **Request Syntax**
  ::

    response = client.list_tags_for_resource(
        resourceArn='string'
    )
    
  :type resourceArn: string
  :param resourceArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the resource. All Amazon Connect resources (instances, queues, flows, routing profiles, etc) have an ARN. To locate the ARN for an instance, for example, see `Find your Amazon Connect instance ID/ARN <https://docs.aws.amazon.com/connect/latest/adminguide/find-instance-arn.html>`__.
