create_opt_out_list
*******************



.. py:method:: PinpointSMSVoiceV2.Client.create_opt_out_list(**kwargs)

  

  Creates a new opt-out list.

   

  If the opt-out list name already exists, an error is returned.

   

  An opt-out list is a list of phone numbers that are opted out, meaning you can't send SMS or voice messages to them. If end user replies with the keyword "STOP," an entry for the phone number is added to the opt-out list. In addition to STOP, your recipients can use any supported opt-out keyword, such as CANCEL or OPTOUT. For a list of supported opt-out keywords, see `SMS opt out <https://docs.aws.amazon.com/pinpoint/latest/userguide/channels-sms-manage.html#channels-sms-manage-optout>`__ in the End User Messaging SMS User Guide.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/pinpoint-sms-voice-v2-2022-03-31/CreateOptOutList>`_
