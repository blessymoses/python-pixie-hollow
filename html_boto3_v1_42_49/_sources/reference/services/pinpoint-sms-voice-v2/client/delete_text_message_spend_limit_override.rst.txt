****************************************
delete_text_message_spend_limit_override
****************************************



.. py:method:: PinpointSMSVoiceV2.Client.delete_text_message_spend_limit_override()

  

  Deletes an account-level monthly spending limit override for sending text messages. Deleting a spend limit override will set the ``EnforcedLimit`` to equal the ``MaxLimit``, which is controlled by Amazon Web Services. For more information on spend limits (quotas) see `Quotas <https://docs.aws.amazon.com/sms-voice/latest/userguide/quotas.html>`__ in the *End User Messaging SMS User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/pinpoint-sms-voice-v2-2022-03-31/DeleteTextMessageSpendLimitOverride>`_  


  **Request Syntax**
  ::

    response = client.delete_text_message_spend_limit_override()
    
    
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'MonthlyLimit': 123
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **MonthlyLimit** *(integer) --* 

        The current monthly limit, in US dollars.
