  .. py:method:: wait(**kwargs)

    Polls :py:meth:`BedrockAgentCoreControl.Client.get_policy` every 2 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/bedrock-agentcore-control-2023-06-05/GetPolicy>`_    


    **Request Syntax**
    ::

      waiter.wait(
          policyEngineId='string',
          policyId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type policyEngineId: string
    :param policyEngineId: **[REQUIRED]** 

      The identifier of the policy engine that manages the policy to be retrieved.

      

    
    :type policyId: string
    :param policyId: **[REQUIRED]** 

      The unique identifier of the policy to be retrieved. This must be a valid policy ID that exists within the specified policy engine.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 2

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60

        

      
    
    
    :returns: None
