  .. py:method:: wait(**kwargs)

    Polls :py:meth:`BedrockAgentCoreControl.Client.get_memory` every 2 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/bedrock-agentcore-control-2023-06-05/GetMemory>`_    


    **Request Syntax**
    ::

      waiter.wait(
          memoryId='string',
          view='full'|'without_decryption',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type memoryId: string
    :param memoryId: **[REQUIRED]** 

      The unique identifier of the memory to retrieve.

      

    
    :type view: string
    :param view: 

      The level of detail to return for the memory.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 2

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60

        

      
    
    
    :returns: None
