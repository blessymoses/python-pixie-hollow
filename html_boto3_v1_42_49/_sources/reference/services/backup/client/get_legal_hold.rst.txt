**************
get_legal_hold
**************



.. py:method:: Backup.Client.get_legal_hold(**kwargs)

  

  This action returns details for a specified legal hold. The details are the body of a legal hold in JSON format, in addition to metadata.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/backup-2018-11-15/GetLegalHold>`_  


  **Request Syntax**
  ::

    response = client.get_legal_hold(
        LegalHoldId='string'
    )
    
  :type LegalHoldId: string
  :param LegalHoldId: **[REQUIRED]** 

    The ID of the legal hold.

    

  
  
  :rtype: dict
  :returns:
