start_insights_refresh
**********************



.. py:method:: EKS.Client.start_insights_refresh(**kwargs)

  

  Initiates an on-demand refresh operation for cluster insights, getting the latest analysis outside of the standard refresh schedule.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/eks-2017-11-01/StartInsightsRefresh>`_  


  **Request Syntax**
  ::

    response = client.start_insights_refresh(
        clusterName='string'
    )
    
  :type clusterName: string
  :param clusterName: **[REQUIRED]** 

    The name of the cluster for the refresh insights operation.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

          'message': 'string',
          'status': 'IN_PROGRESS'|'FAILED'|'COMPLETED'
      }
