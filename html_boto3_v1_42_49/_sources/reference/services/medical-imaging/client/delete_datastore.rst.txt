.. py:method:: HealthImaging.Client.delete_datastore(**kwargs)

  

  Delete a data store.

   

  .. note::

    

    Before a data store can be deleted, you must first delete all image sets within it.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/medical-imaging-2023-07-19/DeleteDatastore>`_  


  **Request Syntax**
  ::

    response = client.delete_datastore(
        datastoreId='string'
    )
    
  :type datastoreId: string
  :param datastoreId: **[REQUIRED]** 

    The data store identifier.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'datastoreId': 'string',
          'datastoreStatus': 'CREATING'|'CREATE_FAILED'|'ACTIVE'|'DELETING'|'DELETED'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **datastoreId** *(string) --* 

        The data store identifier.

        
      

      - **datastoreStatus** *(string) --* 

        The data store status.
