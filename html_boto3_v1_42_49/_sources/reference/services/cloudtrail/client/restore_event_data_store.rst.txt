************************
restore_event_data_store
************************



.. py:method:: CloudTrail.Client.restore_event_data_store(**kwargs)

  

  Restores a deleted event data store specified by ``EventDataStore``, which accepts an event data store ARN. You can only restore a deleted event data store within the seven-day wait period after deletion. Restoring an event data store can take several minutes, depending on the size of the event data store.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudtrail-2013-11-01/RestoreEventDataStore>`_  


  **Request Syntax**
  ::

    response = client.restore_event_data_store(
        EventDataStore='string'
    )
    
  :type EventDataStore: string
  :param EventDataStore: **[REQUIRED]** 

    The ARN (or the ID suffix of the ARN) of the event data store that you want to restore.
