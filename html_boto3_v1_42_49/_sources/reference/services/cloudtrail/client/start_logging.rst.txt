*************
start_logging
*************



.. py:method:: CloudTrail.Client.start_logging(**kwargs)

  

  Starts the recording of Amazon Web Services API calls and log file delivery for a trail. For a trail that is enabled in all Regions, this operation must be called from the Region in which the trail was created. This operation cannot be called on the shadow trails (replicated trails in other Regions) of a trail that is enabled in all Regions.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudtrail-2013-11-01/StartLogging>`_  


  **Request Syntax**
  ::

    response = client.start_logging(
        Name='string'
    )
    
  :type Name: string
  :param Name: **[REQUIRED]** 

    Specifies the name or the CloudTrail ARN of the trail for which CloudTrail logs Amazon Web Services API calls. The following is the format of a trail ARN.

     

    ``arn:aws:cloudtrail:us-east-2:123456789012:trail/MyTrail``
