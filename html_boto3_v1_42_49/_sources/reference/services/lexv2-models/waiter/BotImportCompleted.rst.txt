  .. py:method:: wait(**kwargs)

    Polls :py:meth:`LexModelsV2.Client.describe_import` every 10 seconds until a successful state is reached. An error is raised after 35 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/models.lex.v2-2020-08-07/DescribeImport>`_    


    **Request Syntax**
    ::

      waiter.wait(
          importId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type importId: string
    :param importId: **[REQUIRED]** 

      The unique identifier of the import to describe.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 10

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 35

        

      
    
    
    :returns: None
