start_extension_pack_association
********************************



.. py:method:: DatabaseMigrationService.Client.start_extension_pack_association(**kwargs)

  

  Applies the extension pack to your target database. An extension pack is an add-on module that emulates functions present in a source database that are required when converting objects to the target database.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/dms-2016-01-01/StartExtensionPackAssociation>`_  


  **Request Syntax**
  ::

    response = client.start_extension_pack_association(
        MigrationProjectIdentifier='string'

  :type MigrationProjectIdentifier: string
  :param MigrationProjectIdentifier: **[REQUIRED]** 

    The migration project name or Amazon Resource Name (ARN).
