delete_migration_project
************************



.. py:method:: DatabaseMigrationService.Client.delete_migration_project(**kwargs)

  

  Deletes the specified migration project.

   

  .. note::

    

    The migration project must be closed before you can delete it.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/dms-2016-01-01/DeleteMigrationProject>`_  


  **Request Syntax**
  ::

    response = client.delete_migration_project(
        MigrationProjectIdentifier='string'
    )
    
  :type MigrationProjectIdentifier: string
  :param MigrationProjectIdentifier: **[REQUIRED]** 

    The name or Amazon Resource Name (ARN) of the migration project to delete.

    

  
  
  :rtype: dict
  :returns:
