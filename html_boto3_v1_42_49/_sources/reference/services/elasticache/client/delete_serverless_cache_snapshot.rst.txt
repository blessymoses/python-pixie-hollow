delete_serverless_cache_snapshot
********************************



.. py:method:: ElastiCache.Client.delete_serverless_cache_snapshot(**kwargs)

  

  Deletes an existing serverless cache snapshot. Available for Valkey, Redis OSS and Serverless Memcached only.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/elasticache-2015-02-02/DeleteServerlessCacheSnapshot>`_  


  **Request Syntax**
  ::

    response = client.delete_serverless_cache_snapshot(
        ServerlessCacheSnapshotName='string'
    )
    
  :type ServerlessCacheSnapshotName: string
  :param ServerlessCacheSnapshotName: **[REQUIRED]** 

    Idenfitier of the snapshot to be deleted. Available for Valkey, Redis OSS and Serverless Memcached only.

    

  
  
  :rtype: dict
  :returns:
