create_return_shipping_label

.. py:method:: Snowball.Client.create_return_shipping_label(**kwargs)

  

  Creates a shipping label that will be used to return the Snow device to Amazon Web Services.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/snowball-2016-06-30/CreateReturnShippingLabel>`_  


  **Request Syntax**
  ::

    response = client.create_return_shipping_label(
        JobId='string',
        ShippingOption='SECOND_DAY'|'NEXT_DAY'|'EXPRESS'|'STANDARD'
    )
    
  :type JobId: string
  :param JobId: **[REQUIRED]** 

    The ID for a job that you want to create the return shipping label for; for example, ``JID123e4567-e89b-12d3-a456-426655440000``.

  :param ShippingOption:
