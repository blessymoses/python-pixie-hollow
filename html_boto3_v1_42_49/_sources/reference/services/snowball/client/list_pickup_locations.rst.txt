.. py:method:: Snowball.Client.list_pickup_locations(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/snowball-2016-06-30/ListPickupLocations>`_  


  **Request Syntax**
  ::

    response = client.list_pickup_locations(
        MaxResults=123,
        NextToken='string'
    )
    
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of locations to list per page.

          The address that you want the Snow device(s) associated with a specific job to be shipped to. Addresses are validated at the time of creation. The address you provide must be located within the serviceable area of your region. Although no individual elements of the ``Address`` are required, if the address is invalid or unsupported, then an exception is thrown.

            The unique ID for an address.

          - **Company** *(string) --* 

            The name of the company to receive a Snow device at an address.

            
          

          - **Street1** *(string) --* 

            The first line in a street address that a Snow device is to be delivered to.

            
          

          - **Street2** *(string) --*

          - **Street3** *(string) --*

          - **City** *(string) --*

          - **StateOrProvince** *(string) --*
