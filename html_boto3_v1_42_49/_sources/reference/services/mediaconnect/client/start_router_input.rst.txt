start_router_input
******************



.. py:method:: MediaConnect.Client.start_router_input(**kwargs)

  

  Starts a router input in AWS Elemental MediaConnect.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mediaconnect-2018-11-14/StartRouterInput>`_  


  **Request Syntax**
  ::

    response = client.start_router_input(
        Arn='string'
    )
    
  :type Arn: string
  :param Arn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the router input that you want to start.

    

  
  
  :rtype: dict
  :returns:
