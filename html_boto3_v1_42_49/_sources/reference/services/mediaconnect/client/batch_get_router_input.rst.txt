If a client receives an unknown member it will     set ``SDK_UNKNOWN_MEMBER`` as the top level key,     which maps to the name or tag of the unknown     member. The structure of ``SDK_UNKNOWN_MEMBER`` is     as follows::

                        'SDK_UNKNOWN_MEMBER': {'name': 'UnknownMemberName'}

              - **End** *(datetime) --* 

                The end time of the maintenance window.

                
              

              - **ScheduledTime** *(datetime) --* 

                The date and time when the maintenance window is scheduled to occur.
