*************
delete_bridge
*************



.. py:method:: MediaConnect.Client.delete_bridge(**kwargs)

  

  Deletes a bridge. Before you can delete a bridge, you must stop the bridge.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mediaconnect-2018-11-14/DeleteBridge>`_  


  **Request Syntax**
  ::

    response = client.delete_bridge(
        BridgeArn='string'
    )
    
  :type BridgeArn: string
  :param BridgeArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the bridge that you want to delete.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'BridgeArn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **BridgeArn** *(string) --* 

        The ARN of the deleted bridge.
