  .. py:method:: wait(**kwargs)

    Polls :py:meth:`MediaConnect.Client.get_router_input` every 3 seconds until a successful state is reached. An error is raised after 40 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mediaconnect-2018-11-14/GetRouterInput>`_    


    **Request Syntax**
    ::

      waiter.wait(
          Arn='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type Arn: string
    :param Arn: **[REQUIRED]** 

      The Amazon Resource Name (ARN) of the router input to retrieve information about.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 3

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 40

        

      
    
    
    :returns: None
