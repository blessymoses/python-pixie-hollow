*******************
update_sync_blocker
*******************



.. py:method:: CodeConnections.Client.update_sync_blocker(**kwargs)

  

  Allows you to update the status of a sync blocker, resolving the blocker and allowing syncing to continue.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/codeconnections-2023-12-01/UpdateSyncBlocker>`_  


  **Request Syntax**
  ::

    response = client.update_sync_blocker(
        Id='string',
        SyncType='CFN_STACK_SYNC',
        ResourceName='string',
        ResolvedReason='string'
    )
    
  :type Id: string
  :param Id: **[REQUIRED]** 

    The ID of the sync blocker to be updated.

    

  
  :type SyncType: string
  :param SyncType: **[REQUIRED]** 

    The sync type of the sync blocker to be updated.

    

  
  :type ResourceName: string
  :param ResourceName: **[REQUIRED]** 

    The name of the resource for the sync blocker to be updated.

    

  
  :type ResolvedReason: string
  :param ResolvedReason: **[REQUIRED]** 

    The reason for resolving the sync blocker.

    

  
  
  :rtype: dict
  :returns:
