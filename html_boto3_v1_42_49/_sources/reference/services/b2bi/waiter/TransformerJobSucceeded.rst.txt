  .. py:method:: wait(**kwargs)

    Polls :py:meth:`B2BI.Client.get_transformer_job` every 10 seconds until a successful state is reached. An error is raised after 12 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/b2bi-2022-06-23/GetTransformerJob>`_    


    **Request Syntax**
    ::

      waiter.wait(
          transformerJobId='string',
          transformerId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type transformerJobId: string
    :param transformerJobId: **[REQUIRED]** 

      Specifies the unique, system-generated identifier for a transformer run.

      

    
    :type transformerId: string
    :param transformerId: **[REQUIRED]** 

      Specifies the system-assigned unique identifier for the transformer.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 10

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 12

        

      
    
    
    :returns: None
