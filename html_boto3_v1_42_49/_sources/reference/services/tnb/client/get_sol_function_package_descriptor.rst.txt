get_sol_function_package_descriptor
***********************************



.. py:method:: TelcoNetworkBuilder.Client.get_sol_function_package_descriptor(**kwargs)

  

  Gets a function package descriptor in a function package.

   

  A function package descriptor is a .yaml file in a function package that uses the TOSCA standard to describe how the network function in the function package should run on your network.

   

  A function package is a .zip file in CSAR (Cloud Service Archive) format that contains a network function (an ETSI standard telecommunication application) and function package descriptor that uses the TOSCA standard to describe how the network functions should run on your network.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/tnb-2008-10-21/GetSolFunctionPackageDescriptor>`_  


  **Request Syntax**

        accept='text/plain',
        vnfPkgId='string'
    )

  :type vnfPkgId: string
