.. py:method:: TelcoNetworkBuilder.Client.instantiate_sol_network_instance(**kwargs)

  

  Instantiates a network instance.

   

  A network instance is a single network created in Amazon Web Services TNB that can be deployed and on which life-cycle operations (like terminate, update, and delete) can be performed.

   

  Before you can instantiate a network instance, you have to create a network instance. For more information, see `CreateSolNetworkInstance <https://docs.aws.amazon.com/tnb/latest/APIReference/API_CreateSolNetworkInstance.html>`__.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/tnb-2008-10-21/InstantiateSolNetworkInstance>`_
