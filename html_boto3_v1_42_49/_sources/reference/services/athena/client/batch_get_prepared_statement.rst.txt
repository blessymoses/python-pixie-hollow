****************************
batch_get_prepared_statement
****************************



.. py:method:: Athena.Client.batch_get_prepared_statement(**kwargs)

  

  Returns the details of a single prepared statement or a list of up to 256 prepared statements for the array of prepared statement names that you provide. Requires you to have access to the workgroup to which the prepared statements belong. If a prepared statement cannot be retrieved for the name specified, the statement is listed in ``UnprocessedPreparedStatementNames``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/athena-2017-05-18/BatchGetPreparedStatement>`_  


  **Request Syntax**
  ::

    response = client.batch_get_prepared_statement(
        PreparedStatementNames=[
            'string',
        ],
        WorkGroup='string'
    )
    
  :type PreparedStatementNames: list
  :param PreparedStatementNames: **[REQUIRED]** 

    A list of prepared statement names to return.

    

  
    - *(string) --* 

    

  :type WorkGroup: string
  :param WorkGroup: **[REQUIRED]** 

    The name of the workgroup to which the prepared statements belong.

    

  
  
  :rtype: dict
  :returns:
