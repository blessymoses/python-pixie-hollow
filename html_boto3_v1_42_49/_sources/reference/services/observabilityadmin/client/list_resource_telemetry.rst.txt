.. py:method:: CloudWatchObservabilityAdminService.Client.list_resource_telemetry(**kwargs)

  

  Returns a list of telemetry configurations for Amazon Web Services resources supported by telemetry config. For more information, see `Auditing CloudWatch telemetry configurations <https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/telemetry-config-cloudwatch.html>`__.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/observabilityadmin-2018-05-10/ListResourceTelemetry>`_
