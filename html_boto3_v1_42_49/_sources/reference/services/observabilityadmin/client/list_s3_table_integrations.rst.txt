list_s3_table_integrations
**************************



.. py:method:: CloudWatchObservabilityAdminService.Client.list_s3_table_integrations(**kwargs)

  

  Lists all S3 Table integrations in your account. We recommend using pagination to ensure that the operation returns quickly and successfully.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/observabilityadmin-2018-05-10/ListS3TableIntegrations>`_  


  **Request Syntax**
  ::

    response = client.list_s3_table_integrations(
        MaxResults=123,
        NextToken='string'
    )
    
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of S3 Table integrations to return in a single call.

    

  
  :type NextToken: string
  :param NextToken: 

    The token for the next set of results. A previous call generates this token.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

          'IntegrationSummaries': [

      }
