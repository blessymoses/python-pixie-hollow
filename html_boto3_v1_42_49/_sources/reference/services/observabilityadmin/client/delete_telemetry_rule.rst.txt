delete_telemetry_rule
*********************



.. py:method:: CloudWatchObservabilityAdminService.Client.delete_telemetry_rule(**kwargs)

  

  Deletes a telemetry rule from your account. Any telemetry configurations previously created by the rule will remain but no new resources will be configured by this rule.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/observabilityadmin-2018-05-10/DeleteTelemetryRule>`_  


  **Request Syntax**
  ::

    response = client.delete_telemetry_rule(
        RuleIdentifier='string'
    )
    
  :type RuleIdentifier: string
  :param RuleIdentifier: **[REQUIRED]** 

    The identifier (name or ARN) of the telemetry rule to delete.

    

  
  
  :returns: None
