======
Client
======



.. py:class:: ServiceCatalog.Client

  A low-level client representing AWS Service Catalog
   

  `Service Catalog <http://aws.amazon.com/servicecatalog>`__ enables organizations to create and manage catalogs of IT services that are approved for Amazon Web Services. To get the most out of this documentation, you should be familiar with the terminology discussed in `Service Catalog Concepts <http://docs.aws.amazon.com/servicecatalog/latest/adminguide/what-is_concepts.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('servicecatalog')
