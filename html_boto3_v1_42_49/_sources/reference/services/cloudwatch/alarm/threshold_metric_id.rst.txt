threshold_metric_id
*******************



.. py:attribute:: CloudWatch.Alarm.threshold_metric_id

  

  - *(string) --* 

    In an alarm based on an anomaly detection model, this is the ID of the ``ANOMALY_DETECTION_BAND`` function used as the threshold for the alarm.
