********************
delete_insight_rules
********************



.. py:method:: CloudWatch.Client.delete_insight_rules(**kwargs)

  

  Permanently deletes the specified Contributor Insights rules.

   

  If you create a rule, delete it, and then re-create it with the same name, historical data from the first time the rule was created might not be available.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/monitoring-2010-08-01/DeleteInsightRules>`_  


  **Request Syntax**
  ::

    response = client.delete_insight_rules(
        RuleNames=[
            'string',
        ]
    )
    
  :type RuleNames: list
  :param RuleNames: **[REQUIRED]** 

    An array of the rule names to delete. If you need to find out the names of your rules, use `DescribeInsightRules <https://docs.aws.amazon.com/AmazonCloudWatch/latest/APIReference/API_DescribeInsightRules.html>`__.

    

  
    - *(string) --* 

    

  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'Failures': [
              {
                  'FailureResource': 'string',
                  'ExceptionType': 'string',
                  'FailureCode': 'string',
                  'FailureDescription': 'string'
              },
          ]
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **Failures** *(list) --* 

        An array listing the rules that could not be deleted. You cannot delete built-in rules.

        
        

        - *(dict) --* 

          This array is empty if the API operation was successful for all the rules specified in the request. If the operation could not process one of the rules, the following data is returned for each of those rules.

          
          

          - **FailureResource** *(string) --* 

            The specified rule that could not be deleted.

            
          

          - **ExceptionType** *(string) --* 

            The type of error.

            
          

          - **FailureCode** *(string) --* 

            The code of the error.

            
          

          - **FailureDescription** *(string) --* 

            A description of the error.
