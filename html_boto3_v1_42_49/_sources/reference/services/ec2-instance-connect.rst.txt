======
Client
======



.. py:class:: EC2InstanceConnect.Client

  A low-level client representing AWS EC2 Instance Connect
  

  This is the *Amazon EC2 Instance Connect API Reference*. It provides descriptions, syntax, and usage examples for each of the actions for Amazon EC2 Instance Connect. Amazon EC2 Instance Connect enables system administrators to publish one-time use SSH public keys to EC2, providing users a simple and secure way to connect to their instances.

   

  To view the Amazon EC2 Instance Connect content in the *Amazon EC2 User Guide*, see `Connect to your Linux instance using EC2 Instance Connect <https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/Connect-using-EC2-Instance-Connect.html>`__.

   

  For Amazon EC2 APIs, see the `Amazon EC2 API Reference <https://docs.aws.amazon.com/AWSEC2/latest/APIReference/Welcome.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('ec2-instance-connect')
