***********
update_link
***********



.. py:method:: RTBFabric.Client.update_link(**kwargs)

  

  Updates the configuration of a link between gateways.

   

  Allows you to modify settings and parameters for an existing link.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rtbfabric-2023-05-15/UpdateLink>`_  


  **Request Syntax**
  ::

    response = client.update_link(
        gatewayId='string',
        linkId='string',
        logSettings={
            'applicationLogs': {
                'sampling': {
                    'errorLog': 123.0,
                    'filterLog': 123.0
                }
            }
        }
    )
    
  :type gatewayId: string
  :param gatewayId: **[REQUIRED]** 

    The unique identifier of the gateway.

    

  
  :type linkId: string
  :param linkId: **[REQUIRED]** 

    The unique identifier of the link.

    

  
  :type logSettings: dict
  :param logSettings: 

    Settings for the application logs.

    

  
    - **applicationLogs** *(dict) --* **[REQUIRED]** 

      Describes the configuration of a link application log.

      

    
      - **sampling** *(dict) --* **[REQUIRED]** 

        Describes a link application log sample.

        

      
        - **errorLog** *(float) --* **[REQUIRED]** 

          An error log entry.

          

        
        - **filterLog** *(float) --* **[REQUIRED]** 

          A filter log entry.

          

        
      
    
  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

      {

          'status': 'PENDING_CREATION'|'PENDING_REQUEST'|'REQUESTED'|'ACCEPTED'|'ACTIVE'|'REJECTED'|'FAILED'|'PENDING_DELETION'|'DELETED'|'PENDING_UPDATE'|'PENDING_ISOLATION'|'ISOLATED'|'PENDING_RESTORATION'
      }
