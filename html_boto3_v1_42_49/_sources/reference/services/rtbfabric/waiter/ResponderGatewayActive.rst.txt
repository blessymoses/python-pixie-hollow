  .. py:method:: wait(**kwargs)

    Polls :py:meth:`RTBFabric.Client.get_responder_gateway` every 30 seconds until a successful state is reached. An error is raised after 5 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rtbfabric-2023-05-15/GetResponderGateway>`_    


    **Request Syntax**
    ::

      waiter.wait(
          gatewayId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type gatewayId: string
    :param gatewayId: **[REQUIRED]** 

      The unique identifier of the gateway.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 30

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 5

        

      
    
    
    :returns: None
