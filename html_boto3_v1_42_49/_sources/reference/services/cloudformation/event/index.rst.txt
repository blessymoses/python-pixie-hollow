Event

.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: CloudFormation.Event(id)

  A resource representing an AWS CloudFormation Event::

    
    import boto3
    
    cloudformation = boto3.resource('cloudformation')
    event = cloudformation.Event('id')

  :type id: string
  :param id: The Event's id identifier. This **must** be set.
