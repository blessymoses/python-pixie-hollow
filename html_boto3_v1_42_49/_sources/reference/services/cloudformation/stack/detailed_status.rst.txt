***************
detailed_status
***************



.. py:attribute:: CloudFormation.Stack.detailed_status

  

  - *(string) --* 

    The detailed status of the resource or stack. If ``CONFIGURATION_COMPLETE`` is present, the resource or resource configuration phase has completed and the stabilization of the resources is in progress. The StackSets ``CONFIGURATION_COMPLETE`` when all of the resources in the stack have reached that event. For more information, see `Understand CloudFormation stack creation events <https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/stack-resource-configuration-complete.html>`__ in the *CloudFormation User Guide*.
