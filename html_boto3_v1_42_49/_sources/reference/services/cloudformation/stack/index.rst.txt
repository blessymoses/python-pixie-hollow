Stack
=====



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: CloudFormation.Stack(name)

  A resource representing an AWS CloudFormation Stack::

    
    import boto3
    
    cloudformation = boto3.resource('cloudformation')
    stack = cloudformation.Stack('name')

  :type name: string
  :param name: The Stack's name identifier. This **must** be set.
