*************
delete_layout
*************



.. py:method:: ConnectCases.Client.delete_layout(**kwargs)

  

  Deletes a layout from a cases template. You can delete up to 100 layouts per domain.

   

  After a layout is deleted:

   

  
  * You can still retrieve the layout by calling ``GetLayout``.
   
  * You cannot update a deleted layout by calling ``UpdateLayout``; it throws a ``ValidationException``.
   
  * Deleted layouts are not included in the ``ListLayouts`` response.
  

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/connectcases-2022-10-03/DeleteLayout>`_  


  **Request Syntax**
  ::

    response = client.delete_layout(
        domainId='string',
        layoutId='string'
    )
    
  :type domainId: string
  :param domainId: **[REQUIRED]** 

    The unique identifier of the Cases domain.

    

  
  :type layoutId: string
  :param layoutId: **[REQUIRED]** 

    The unique identifier of the layout.
