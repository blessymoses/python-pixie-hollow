list_playback_restriction_policies

.. py:method:: IVS.Client.list_playback_restriction_policies(**kwargs)

  

  Gets summary information about playback restriction policies.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ivs-2020-07-14/ListPlaybackRestrictionPolicies>`_  


  **Request Syntax**
  ::

    response = client.list_playback_restriction_policies(
        maxResults=123,
        nextToken='string'
    )
    
  :type maxResults: integer
  :param maxResults: 

    Maximum number of policies to return. Default: 1.

    

  
  :type nextToken: string
  :param nextToken: 

    The first policy to retrieve. This is used for pagination; see the ``nextToken`` response field.

    

  
  
  :rtype: dict
  :returns:
