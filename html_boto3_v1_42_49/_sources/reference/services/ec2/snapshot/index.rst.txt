========
Snapshot
========



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: EC2.Snapshot(id)

  A resource representing an Amazon Elastic Compute Cloud (EC2) Snapshot::

    
    import boto3
    
    ec2 = boto3.resource('ec2')
    snapshot = ec2.Snapshot('id')

  :type id: string
  :param id: The Snapshot's id identifier. This **must** be set.
