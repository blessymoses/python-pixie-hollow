******************
source_instance_id
******************



.. py:attribute:: EC2.Image.source_instance_id

  

  - *(string) --* 

    The ID of the instance that the AMI was created from if the AMI was created using `CreateImage <https://docs.aws.amazon.com/AWSEC2/latest/APIReference/API_CreateImage.html>`__. This field only appears if the AMI was created using CreateImage.
