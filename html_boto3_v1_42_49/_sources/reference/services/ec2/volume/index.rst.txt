Volume
======



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: EC2.Volume(id)

  A resource representing an Amazon Elastic Compute Cloud (EC2) Volume::

    
    import boto3
    
    ec2 = boto3.resource('ec2')
    volume = ec2.Volume('id')

  :type id: string
  :param id: The Volume's id identifier. This **must** be set.
