***************
public_dns_name
***************



.. py:attribute:: EC2.Instance.public_dns_name

  

  - *(string) --* 

    The public DNS name assigned to the instance. This name is not available until the instance enters the ``running`` state. This name is only available if you've enabled DNS hostnames for your VPC. The format of this name depends on the `public hostname type <https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/hostname-types.html#public-hostnames>`__.
