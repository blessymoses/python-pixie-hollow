*******************
hibernation_options
*******************



.. py:attribute:: EC2.Instance.hibernation_options

  

  - *(dict) --* 

    Indicates whether the instance is enabled for hibernation.

    
    

    - **Configured** *(boolean) --* 

      If ``true``, your instance is enabled for hibernation; otherwise, it is not enabled for hibernation.
