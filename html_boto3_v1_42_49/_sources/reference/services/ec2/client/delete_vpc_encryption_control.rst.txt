delete_vpc_encryption_control
*****************************



.. py:method:: EC2.Client.delete_vpc_encryption_control(**kwargs)

  

  Deletes a VPC Encryption Control configuration. This removes the encryption policy enforcement from the specified VPC.

   

  For more information, see `Enforce VPC encryption in transit <https://docs.aws.amazon.com/vpc/latest/userguide/vpc-encryption-controls.html>`__ in the *Amazon VPC User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DeleteVpcEncryptionControl>`_  


  **Request Syntax**
  ::

    response = client.delete_vpc_encryption_control(
        DryRun=True|False,
        VpcEncryptionControlId='string'
    )
    
  :type DryRun: boolean
  :param DryRun: 

    Checks whether you have the required permissions for the action, without actually making the request, and provides an error response. If you have the required permissions, the error response is ``DryRunOperation``. Otherwise, it is ``UnauthorizedOperation``.

    

  
  :type VpcEncryptionControlId: string
  :param VpcEncryptionControlId: **[REQUIRED]** 

    The ID of the VPC Encryption Control resource to delete.
