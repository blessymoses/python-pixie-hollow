describe_fast_snapshot_restores
*******************************



.. py:method:: EC2.Client.describe_fast_snapshot_restores(**kwargs)

  

  Describes the state of fast snapshot restores for your snapshots.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DescribeFastSnapshotRestores>`_  


  **Request Syntax**

        Filters=[
            {
                'Name': 'string',
                'Values': [
                    'string',
                ]
            },
        ],
        MaxResults=123,
        NextToken='string',
        DryRun=True|False
    )
    
  :type Filters: list
  :param Filters: 

    The filters. The possible values are:

     

    
    * ``availability-zone``: The Availability Zone of the snapshot. For example, ``us-east-2a``.
     
    * ``availability-zone-id``: The ID of the Availability Zone of the snapshot. For example, ``use2-az1``.
     
    * ``owner-id``: The ID of the Amazon Web Services account that enabled fast snapshot restore on the snapshot.
     
    * ``snapshot-id``: The ID of the snapshot.
     
    * ``state``: The state of fast snapshot restores for the snapshot ( ``enabling`` | ``optimizing`` | ``enabled`` | ``disabling`` | ``disabled``).
