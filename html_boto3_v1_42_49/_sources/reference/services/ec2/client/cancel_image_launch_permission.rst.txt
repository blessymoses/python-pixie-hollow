******************************
cancel_image_launch_permission
******************************



.. py:method:: EC2.Client.cancel_image_launch_permission(**kwargs)

  

  Removes your Amazon Web Services account from the launch permissions for the specified AMI. For more information, see `Cancel having an AMI shared with your Amazon Web Services account <https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/cancel-sharing-an-AMI.html>`__ in the *Amazon EC2 User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/CancelImageLaunchPermission>`_  


  **Request Syntax**
  ::

    response = client.cancel_image_launch_permission(
        ImageId='string',
        DryRun=True|False
    )
    
  :type ImageId: string
  :param ImageId: **[REQUIRED]** 

    The ID of the AMI that was shared with your Amazon Web Services account.

    

  
  :type DryRun: boolean
  :param DryRun: 

    Checks whether you have the required permissions for the action, without actually making the request, and provides an error response. If you have the required permissions, the error response is ``DryRunOperation``. Otherwise, it is ``UnauthorizedOperation``.

    

  
  
  :rtype: dict
  :returns:

      {
          'Return': True|False
      }
