delete_ipam_prefix_list_resolver
********************************



.. py:method:: EC2.Client.delete_ipam_prefix_list_resolver(**kwargs)

  

  Deletes an IPAM prefix list resolver. Before deleting a resolver, you must first delete all resolver targets associated with it.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DeleteIpamPrefixListResolver>`_  


  **Request Syntax**
  ::

    response = client.delete_ipam_prefix_list_resolver(
        DryRun=True|False,
        IpamPrefixListResolverId='string'
    )
    
  :type DryRun: boolean
  :param DryRun: 

    A check for whether you have the required permissions for the action without actually making the request and provides an error response. If you have the required permissions, the error response is ``DryRunOperation``. Otherwise, it is ``UnauthorizedOperation``.

    

  
  :type IpamPrefixListResolverId: string
