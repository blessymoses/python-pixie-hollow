.. py:method:: EC2.Client.delete_traffic_mirror_target(**kwargs)

  

  Deletes the specified Traffic Mirror target.

   

  You cannot delete a Traffic Mirror target that is in use by a Traffic Mirror session.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DeleteTrafficMirrorTarget>`_  


  **Request Syntax**
  ::

        TrafficMirrorTargetId='string',
        DryRun=True|False
    )
    
  :type TrafficMirrorTargetId: string
  :param TrafficMirrorTargetId: **[REQUIRED]** 

    The ID of the Traffic Mirror target.

    

  
  :type DryRun: boolean
  :param DryRun:
