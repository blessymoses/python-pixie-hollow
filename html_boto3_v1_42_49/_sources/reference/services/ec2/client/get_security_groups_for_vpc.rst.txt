get_security_groups_for_vpc
***************************



.. py:method:: EC2.Client.get_security_groups_for_vpc(**kwargs)

  

  Gets security groups that can be associated by the Amazon Web Services account making the request with network interfaces in the specified VPC.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/GetSecurityGroupsForVpc>`_  


  **Request Syntax**
  ::

    response = client.get_security_groups_for_vpc(
        VpcId='string',
        NextToken='string',
        MaxResults=123,
        Filters=[
            {
                'Name': 'string',
                'Values': [
                    'string',
                ]
            },
        ],
        DryRun=True|False
    )
    
  :type VpcId: string
  :param VpcId: **[REQUIRED]** 

    The VPC ID where the security group can be used.

    

  
  :type NextToken: string
  :param NextToken:

  :type MaxResults: integer

  :type Filters: list
  :param Filters:

    * ``primary-vpc-id``: The VPC ID in which the security group was created.
