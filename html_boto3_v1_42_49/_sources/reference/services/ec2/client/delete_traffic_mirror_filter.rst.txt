.. py:method:: EC2.Client.delete_traffic_mirror_filter(**kwargs)

  

  Deletes the specified Traffic Mirror filter.

   

  You cannot delete a Traffic Mirror filter that is in use by a Traffic Mirror session.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DeleteTrafficMirrorFilter>`_
