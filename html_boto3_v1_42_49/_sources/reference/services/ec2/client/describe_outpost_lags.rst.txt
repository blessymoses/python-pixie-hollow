describe_outpost_lags
*********************



.. py:method:: EC2.Client.describe_outpost_lags(**kwargs)

  

  Describes the Outposts link aggregation groups (LAGs).

   

  .. note::

    

    LAGs are only available for second-generation Outposts racks at this time.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DescribeOutpostLags>`_  


  **Request Syntax**
  ::

    response = client.describe_outpost_lags(
        OutpostLagIds=[
            'string',
        ],
        Filters=[
            {
                'Name': 'string',
                'Values': [
                    'string',
                ]
            },
        ],
        MaxResults=123,
        NextToken='string',
        DryRun=True|False
    )
    
  :type OutpostLagIds: list
  :param OutpostLagIds: 

    The IDs of the Outpost LAGs.

    * ``peer-bgp-asn`` - The peer BGP ASN.
