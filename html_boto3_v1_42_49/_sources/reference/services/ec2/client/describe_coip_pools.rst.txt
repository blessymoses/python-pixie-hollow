describe_coip_pools
*******************



.. py:method:: EC2.Client.describe_coip_pools(**kwargs)

  

  Describes the specified customer-owned address pools or all of your customer-owned address pools.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DescribeCoipPools>`_

            'string',

        Filters=[
            {
                'Name': 'string',
                'Values': [
                    'string',
                ]
            },
        ],
        MaxResults=123,
        NextToken='string',
        DryRun=True|False
    )
