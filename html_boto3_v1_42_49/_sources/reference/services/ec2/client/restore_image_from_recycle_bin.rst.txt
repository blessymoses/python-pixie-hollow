restore_image_from_recycle_bin
******************************



.. py:method:: EC2.Client.restore_image_from_recycle_bin(**kwargs)

  

  Restores an AMI from the Recycle Bin. For more information, see `Recover deleted Amazon EBS snapshots and EBS-back AMIs with Recycle Bin <https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/recycle-bin.html>`__ in the *Amazon EC2 User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/RestoreImageFromRecycleBin>`_  


  **Request Syntax**
  ::

    response = client.restore_image_from_recycle_bin(
        ImageId='string',
        DryRun=True|False
    )
    
  :type ImageId: string
  :param ImageId: **[REQUIRED]** 

    The ID of the AMI to restore.

    

  
  :type DryRun: boolean
  :param DryRun: 

    Checks whether you have the required permissions for the action, without actually making the request, and provides an error response. If you have the required permissions, the error response is ``DryRunOperation``. Otherwise, it is ``UnauthorizedOperation``.
