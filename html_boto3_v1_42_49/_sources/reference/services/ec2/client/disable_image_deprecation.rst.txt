disable_image_deprecation
*************************



.. py:method:: EC2.Client.disable_image_deprecation(**kwargs)

  

  Cancels the deprecation of the specified AMI.

   

  For more information, see `Deprecate an Amazon EC2 AMI <https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/ami-deprecate.html>`__ in the *Amazon EC2 User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DisableImageDeprecation>`_  


  **Request Syntax**
  ::

    response = client.disable_image_deprecation(
        ImageId='string',
        DryRun=True|False
    )
    
  :type ImageId: string
  :param ImageId: **[REQUIRED]** 

    The ID of the AMI.

    

  
  :type DryRun: boolean
  :param DryRun:
