  .. py:method:: wait(**kwargs)

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ec2-2016-11-15/DescribeConversionTasks>`_

              'Delay': 123,
              'MaxAttempts': 123
          }
      )
