Subnet

.. py:class:: EC2.Subnet(id)

  A resource representing an Amazon Elastic Compute Cloud (EC2) Subnet::

    
    import boto3
    
    ec2 = boto3.resource('ec2')
    subnet = ec2.Subnet('id')

  :type id: string
  :param id: The Subnet's id identifier. This **must** be set.
