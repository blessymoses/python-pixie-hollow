.. py:attribute:: EC2.Subnet.type

  

  - *(string) --* 

    Indicates if this is a subnet used with Amazon Elastic VMware Service (EVS). Possible values are ``Elastic VMware Service`` or no value. For more information about Amazon EVS, see `Amazon Elastic VMware Service API Reference <https://docs.aws.amazon.com/evs/latest/APIReference/Welcome.html>`__.
