list_tokens

.. py:method:: LicenseManager.Client.list_tokens(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/license-manager-2018-08-01/ListTokens>`_  


  **Request Syntax**
  ::

    response = client.list_tokens(
        TokenIds=[
            'string',

        Filters=[
            {
                'Name': 'string',

                    'string',
                ]
            },
        ],
        NextToken='string',
        MaxResults=123
    )

        - *(string) --*
