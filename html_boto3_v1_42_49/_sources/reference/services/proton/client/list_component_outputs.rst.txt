list_component_outputs
**********************



.. py:method:: Proton.Client.list_component_outputs(**kwargs)

  

  Get a list of component Infrastructure as Code (IaC) outputs.

   

  For more information about components, see `Proton components <https://docs.aws.amazon.com/proton/latest/userguide/ag-components.html>`__ in the *Proton User Guide*.

  

  .. danger::

        This operation is deprecated and may not function as expected. This operation should not be used going forward and is only kept for the purpose of backwards compatiblity.


  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/proton-2020-07-20/ListComponentOutputs>`_  


  **Request Syntax**
  ::

    response = client.list_component_outputs(
        componentName='string',
        deploymentId='string',
        nextToken='string'
    )
    
  :type componentName: string
  :param componentName: **[REQUIRED]** 

    The name of the component whose outputs you want.

    

  
  :type deploymentId: string
  :param deploymentId: 

    The ID of the deployment whose outputs you want.

    

  
  :type nextToken: string
  :param nextToken: 

    A token that indicates the location of the next output in the array of outputs, after the list of outputs that was previously requested.

    

  
  
  :rtype: dict
  :returns:
