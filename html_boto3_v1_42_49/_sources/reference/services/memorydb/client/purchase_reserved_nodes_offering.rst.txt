purchase_reserved_nodes_offering
********************************



.. py:method:: MemoryDB.Client.purchase_reserved_nodes_offering(**kwargs)

  

  Allows you to purchase a reserved node offering. Reserved nodes are not eligible for cancellation and are non-refundable.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/memorydb-2021-01-01/PurchaseReservedNodesOffering>`_  


  **Request Syntax**
  ::

    response = client.purchase_reserved_nodes_offering(
        ReservedNodesOfferingId='string',
        ReservationId='string',
        NodeCount=123,
        Tags=[
            {
                'Key': 'string',
                'Value': 'string'
            },
        ]
    )
    
  :type ReservedNodesOfferingId: string
  :param ReservedNodesOfferingId: **[REQUIRED]** 

    The ID of the reserved node offering to purchase.

    

  
  :type ReservationId: string
  :param ReservationId: 

    A customer-specified identifier to track this reservation.

    

  
  :type NodeCount: integer
  :param NodeCount: 

    The number of node instances to reserve.

    

  
  :type Tags: list
  :param Tags:
