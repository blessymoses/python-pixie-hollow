*******************
delete_subnet_group
*******************



.. py:method:: MemoryDB.Client.delete_subnet_group(**kwargs)

  

  Deletes a subnet group. You cannot delete a default subnet group or one that is associated with any clusters.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/memorydb-2021-01-01/DeleteSubnetGroup>`_  


  **Request Syntax**
  ::

    response = client.delete_subnet_group(
        SubnetGroupName='string'
    )
    
  :type SubnetGroupName: string
  :param SubnetGroupName: **[REQUIRED]** 

    The name of the subnet group to delete.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**
