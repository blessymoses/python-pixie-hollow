get_data_set_export_task

.. py:method:: MainframeModernization.Client.get_data_set_export_task(**kwargs)

  

  Gets the status of a data set import task initiated with the  CreateDataSetExportTask operation.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/m2-2021-04-28/GetDataSetExportTask>`_  


  **Request Syntax**
  ::

    response = client.get_data_set_export_task(
        applicationId='string',
        taskId='string'
    )
    
  :type applicationId: string
  :param applicationId: **[REQUIRED]** 

    The application identifier.

    

  
  :type taskId: string
  :param taskId: **[REQUIRED]** 

    The task identifier returned by the  CreateDataSetExportTask operation.

    

  
  
  :rtype: dict
  :returns:

      {
