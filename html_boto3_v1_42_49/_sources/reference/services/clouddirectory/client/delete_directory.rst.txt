****************
delete_directory
****************



.. py:method:: CloudDirectory.Client.delete_directory(**kwargs)

  

  Deletes a directory. Only disabled directories can be deleted. A deleted directory cannot be undone. Exercise extreme caution when deleting directories.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/clouddirectory-2017-01-11/DeleteDirectory>`_  


  **Request Syntax**
  ::

    response = client.delete_directory(
        DirectoryArn='string'
    )
    
  :type DirectoryArn: string
  :param DirectoryArn: **[REQUIRED]** 

    The ARN of the directory to delete.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'DirectoryArn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **DirectoryArn** *(string) --* 

        The ARN of the deleted directory.
