tag_resource
************



.. py:method:: PartnerCentralBenefits.Client.tag_resource(**kwargs)

  

  Adds or updates tags for a specified resource.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/partnercentral-benefits-2018-05-10/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        resourceArn='string',
        tags=[
            {
                'Key': 'string',
                'Value': 'string'
            },
        ]
    )
    
  :type resourceArn: string
  :param resourceArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the resource to add tags to.

    

  
  :type tags: list
  :param tags: **[REQUIRED]** 

    A list of key-value pairs to add as tags to the resource.

    

  
    - *(dict) --* 

      Represents a key-value pair used for categorizing and organizing AWS resources.

      

    
      - **Key** *(string) --* **[REQUIRED]** 

        The tag key, which acts as a category or label for the tag.

        

      
      - **Value** *(string) --* **[REQUIRED]** 

        The tag value, which provides additional information or context for the tag key.
