======
Client
======



.. py:class:: ChimeSDKMessaging.Client

  A low-level client representing Amazon Chime SDK Messaging
  

  The Amazon Chime SDK messaging APIs in this section allow software developers to send and receive messages in custom messaging applications. These APIs depend on the frameworks provided by the Amazon Chime SDK identity APIs. For more information about the messaging APIs, see `Amazon Chime SDK messaging <https://docs.aws.amazon.com/chime-sdk/latest/APIReference/API_Operations_Amazon_Chime_SDK_Messaging.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('chime-sdk-messaging')
