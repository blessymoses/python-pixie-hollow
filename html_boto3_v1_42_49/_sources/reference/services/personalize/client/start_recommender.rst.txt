*****************
start_recommender
*****************



.. py:method:: Personalize.Client.start_recommender(**kwargs)

  

  Starts a recommender that is INACTIVE. Starting a recommender does not create any new models, but resumes billing and automatic retraining for the recommender.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/personalize-2018-05-22/StartRecommender>`_  


  **Request Syntax**
  ::

    response = client.start_recommender(
        recommenderArn='string'
    )
    
  :type recommenderArn: string
  :param recommenderArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the recommender to start.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'recommenderArn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **recommenderArn** *(string) --* 

        The Amazon Resource Name (ARN) of the recommender you started.
