****************
stop_recommender
****************



.. py:method:: Personalize.Client.stop_recommender(**kwargs)

  

  Stops a recommender that is ACTIVE. Stopping a recommender halts billing and automatic retraining for the recommender.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/personalize-2018-05-22/StopRecommender>`_  


  **Request Syntax**
  ::

    response = client.stop_recommender(
        recommenderArn='string'
    )
    
  :type recommenderArn: string
  :param recommenderArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the recommender to stop.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'recommenderArn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **recommenderArn** *(string) --* 

        The Amazon Resource Name (ARN) of the recommender you stopped.
