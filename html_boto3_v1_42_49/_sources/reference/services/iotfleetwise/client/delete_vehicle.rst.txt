delete_vehicle
**************



.. py:method:: IoTFleetWise.Client.delete_vehicle(**kwargs)

  

  Deletes a vehicle and removes it from any campaigns.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iotfleetwise-2021-06-17/DeleteVehicle>`_  


  **Request Syntax**
  ::

    response = client.delete_vehicle(
        vehicleName='string'
    )
    
  :type vehicleName: string
  :param vehicleName: **[REQUIRED]** 

    The ID of the vehicle to delete.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'vehicleName': 'string',
          'arn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **vehicleName** *(string) --* 

        The ID of the deleted vehicle.

        
      

      - **arn** *(string) --* 

        The Amazon Resource Name (ARN) of the deleted vehicle.
