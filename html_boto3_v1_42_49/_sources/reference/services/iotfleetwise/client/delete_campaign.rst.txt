***************
delete_campaign
***************



.. py:method:: IoTFleetWise.Client.delete_campaign(**kwargs)

  

  Deletes a data collection campaign. Deleting a campaign suspends all data collection and removes it from any vehicles.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iotfleetwise-2021-06-17/DeleteCampaign>`_  


  **Request Syntax**
  ::

    response = client.delete_campaign(
        name='string'
    )
    
  :type name: string
  :param name: **[REQUIRED]** 

    The name of the campaign to delete.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'name': 'string',
          'arn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **name** *(string) --* 

        The name of the deleted campaign.

        
      

      - **arn** *(string) --* 

        The Amazon Resource Name (ARN) of the deleted campaign.

         

        .. note::

          

          The ARN isn’t returned if a campaign doesn’t exist.
