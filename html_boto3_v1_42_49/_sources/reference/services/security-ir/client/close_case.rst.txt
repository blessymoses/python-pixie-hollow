**********
close_case
**********



.. py:method:: SecurityIncidentResponse.Client.close_case(**kwargs)

  

  Closes an existing case.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/security-ir-2018-05-10/CloseCase>`_  


  **Request Syntax**
  ::

    response = client.close_case(
        caseId='string'
    )
    
  :type caseId: string
  :param caseId: **[REQUIRED]** 

    Required element used in combination with CloseCase to identify the case ID to close.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'caseStatus': 'Submitted'|'Acknowledged'|'Detection and Analysis'|'Containment, Eradication and Recovery'|'Post-incident Activities'|'Ready to Close'|'Closed',
          'closedDate': datetime(2015, 1, 1)
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **caseStatus** *(string) --* 

        A response element providing responses for requests to CloseCase. This element responds ``Closed`` if successful.

        
      

      - **closedDate** *(datetime) --* 

        A response element providing responses for requests to CloseCase. This element responds with the ISO-8601 formatted timestamp of the moment when the case was closed.
