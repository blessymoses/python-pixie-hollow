cancel_gremlin_query
********************



.. py:method:: NeptuneData.Client.cancel_gremlin_query(**kwargs)

  

  Cancels a Gremlin query. See `Gremlin query cancellation <https://docs.aws.amazon.com/neptune/latest/userguide/gremlin-api-status-cancel.html>`__ for more information.

   

  When invoking this operation in a Neptune cluster that has IAM authentication enabled, the IAM user or role making the request must have a policy attached that allows the `neptune-db\:CancelQuery <https://docs.aws.amazon.com/neptune/latest/userguide/iam-dp-actions.html#cancelquery>`__ IAM action in that cluster.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/neptunedata-2023-08-01/CancelGremlinQuery>`_  


  **Request Syntax**
  ::

    response = client.cancel_gremlin_query(
        queryId='string'
    )
    
  :type queryId: string
  :param queryId: **[REQUIRED]** 

    The unique identifier that identifies the query to be canceled.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'status': 'string'
      }

        The status of the cancelation
