put_voice_connector_logging_configuration
*****************************************



.. py:method:: ChimeSDKVoice.Client.put_voice_connector_logging_configuration(**kwargs)

  

  Updates a Voice Connector's logging configuration.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/chime-sdk-voice-2022-08-03/PutVoiceConnectorLoggingConfiguration>`_  


  **Request Syntax**
  ::

    response = client.put_voice_connector_logging_configuration(
        VoiceConnectorId='string',
        LoggingConfiguration={
            'EnableSIPLogs': True|False,
            'EnableMediaMetricLogs': True|False
        }
    )
    
  :type VoiceConnectorId: string
  :param VoiceConnectorId: **[REQUIRED]** 

    The Voice Connector ID.

    

  
  :type LoggingConfiguration: dict
  :param LoggingConfiguration: **[REQUIRED]** 

    The logging configuration being updated.

    

  
    - **EnableSIPLogs** *(boolean) --* 

      Boolean that enables sending SIP message logs to Amazon CloudWatch.

      

    
    - **EnableMediaMetricLogs** *(boolean) --* 

      Enables or disables media metrics logging.

      

    
  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'LoggingConfiguration': {
              'EnableSIPLogs': True|False,
              'EnableMediaMetricLogs': True|False
          }
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **LoggingConfiguration** *(dict) --* 

        The updated logging configuration.

        
        

        - **EnableSIPLogs** *(boolean) --* 

          Boolean that enables sending SIP message logs to Amazon CloudWatch.

          
        

        - **EnableMediaMetricLogs** *(boolean) --* 

          Enables or disables media metrics logging.
