**************************************
get_voice_connector_termination_health
**************************************



.. py:method:: ChimeSDKVoice.Client.get_voice_connector_termination_health(**kwargs)

  

  Retrieves information about the last time a ``SIP OPTIONS`` ping was received from your SIP infrastructure for the specified Amazon Chime SDK Voice Connector.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/chime-sdk-voice-2022-08-03/GetVoiceConnectorTerminationHealth>`_  


  **Request Syntax**
  ::

    response = client.get_voice_connector_termination_health(
        VoiceConnectorId='string'
    )
    
  :type VoiceConnectorId: string
  :param VoiceConnectorId: **[REQUIRED]** 

    The Voice Connector ID.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'TerminationHealth': {
              'Timestamp': datetime(2015, 1, 1),
              'Source': 'string'
          }
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **TerminationHealth** *(dict) --* 

        The termination health details.

        
        

        - **Timestamp** *(datetime) --* 

          The timestamp, in ISO 8601 format.

          
        

        - **Source** *(string) --* 

          The source IP address.
