.. py:method:: ChimeSDKVoice.Client.get_speaker_search_task(**kwargs)

  **Request Syntax**

        VoiceConnectorId='string',
        SpeakerSearchTaskId='string'

  :param VoiceConnectorId: **[REQUIRED]** 

    The Voice Connector ID.

    

  
  :type SpeakerSearchTaskId: string
