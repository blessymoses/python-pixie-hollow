tag_resource
************



.. py:method:: ChimeSDKVoice.Client.tag_resource(**kwargs)

  

  Adds a tag to the specified resource.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/chime-sdk-voice-2022-08-03/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        ResourceARN='string',
        Tags=[
            {
                'Key': 'string',
                'Value': 'string'
            },
        ]
    )
    
  :type ResourceARN: string
  :param ResourceARN: **[REQUIRED]** 

    The ARN of the resource being tagged.

    

  
  :type Tags: list
  :param Tags: **[REQUIRED]** 

    A list of the tags being added to the resource.

    

  
    - *(dict) --* 

      Describes a tag applied to a resource.

      

    
      - **Key** *(string) --* **[REQUIRED]** 

        The tag's key.

        

      
      - **Value** *(string) --* **[REQUIRED]** 

        The tag's value.
