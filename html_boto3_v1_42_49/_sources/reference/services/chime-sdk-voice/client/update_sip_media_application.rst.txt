.. py:method:: ChimeSDKVoice.Client.update_sip_media_application(**kwargs)

  **Request Syntax**

        SipMediaApplicationId='string',
        Name='string',
        Endpoints=[
            {
                'LambdaArn': 'string'
            },
        ]
    )
    
  :type SipMediaApplicationId: string
  :param SipMediaApplicationId: **[REQUIRED]** 

    The SIP media application ID.

    

  
  :type Name: string
  :param Name:
