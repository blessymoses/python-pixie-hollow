******
delete
******



.. py:method:: IAM.GroupPolicy.delete()

  

  Deletes the specified inline policy that is embedded in the specified IAM group.

   

  A group can also have managed policies attached to it. To detach a managed policy from a group, use `DetachGroupPolicy <https://docs.aws.amazon.com/IAM/latest/APIReference/API_DetachGroupPolicy.html>`__. For more information about policies, refer to `Managed policies and inline policies <https://docs.aws.amazon.com/IAM/latest/UserGuide/policies-managed-vs-inline.html>`__ in the *IAM User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iam-2010-05-08/DeleteGroupPolicy>`_  


  **Request Syntax**
  ::

    response = group_policy.delete()
    
    
  
  :returns: None
