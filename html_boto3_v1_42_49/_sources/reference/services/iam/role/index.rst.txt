====
Role
====



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: IAM.Role(name)

  A resource representing an AWS Identity and Access Management (IAM) Role::

    
    import boto3
    
    iam = boto3.resource('iam')
    role = iam.Role('name')

  :type name: string
  :param name: The Role's name identifier. This **must** be set.
