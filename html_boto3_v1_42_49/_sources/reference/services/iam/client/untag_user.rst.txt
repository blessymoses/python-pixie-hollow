**********
untag_user
**********



.. py:method:: IAM.Client.untag_user(**kwargs)

  

  Removes the specified tags from the user. For more information about tagging, see `Tagging IAM resources <https://docs.aws.amazon.com/IAM/latest/UserGuide/id_tags.html>`__ in the *IAM User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iam-2010-05-08/UntagUser>`_  


  **Request Syntax**
  ::

    response = client.untag_user(
        UserName='string',
        TagKeys=[
            'string',
        ]
    )
    
  :type UserName: string
  :param UserName: **[REQUIRED]** 

    The name of the IAM user from which you want to remove tags.

     

    This parameter allows (through its `regex pattern <http://wikipedia.org/wiki/regex>`__) a string of characters consisting of upper and lowercase alphanumeric characters with no spaces. You can also include any of the following characters: _+=,.@-

    

  
  :type TagKeys: list
  :param TagKeys: **[REQUIRED]** 

    A list of key names as a simple array of strings. The tags with matching keys are removed from the specified user.

    

  
    - *(string) --* 

    

  
  :returns: None
  **Exceptions**
  
  *   :py:class:`IAM.Client.exceptions.NoSuchEntityException`

  
  *   :py:class:`IAM.Client.exceptions.ConcurrentModificationException`

  
  *   :py:class:`IAM.Client.exceptions.ServiceFailureException`

  

  **Examples**

  The following example shows how to remove tags that are attached to a user named 'anika'.
  ::

    response = client.untag_user(
        TagKeys=[
            'Dept',
        ],
        UserName='anika',
    )
    
    print(response)
