*******************
list_saml_providers
*******************



.. py:method:: IAM.Client.list_saml_providers()

  

  Lists the SAML provider resource objects defined in IAM in the account. IAM resource-listing operations return a subset of the available attributes for the resource. For example, this operation does not return tags, even though they are an attribute of the returned object. To view all of the information for a SAML provider, see `GetSAMLProvider <https://docs.aws.amazon.com/IAM/latest/APIReference/API_GetSAMLProvider.html>`__.

   

  .. warning::

     

    This operation requires `Signature Version 4 <https://docs.aws.amazon.com/general/latest/gr/signature-version-4.html>`__.

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iam-2010-05-08/ListSAMLProviders>`_  


  **Request Syntax**
  ::

    response = client.list_saml_providers()
    
    
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'SAMLProviderList': [
              {
                  'Arn': 'string',
                  'ValidUntil': datetime(2015, 1, 1),
                  'CreateDate': datetime(2015, 1, 1)
              },
          ]
      }
