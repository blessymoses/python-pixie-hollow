=====
Group
=====



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: IAM.Group(name)

  A resource representing an AWS Identity and Access Management (IAM) Group::

    
    import boto3
    
    iam = boto3.resource('iam')
    group = iam.Group('name')

  :type name: string
  :param name: The Group's name identifier. This **must** be set.
