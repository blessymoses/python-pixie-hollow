  .. py:method:: wait(**kwargs)

    Polls :py:meth:`Omics.Client.get_workflow` every 3 seconds until a successful state is reached. An error is raised after 10 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/omics-2022-11-28/GetWorkflow>`_

              'Delay': 123,
              'MaxAttempts': 123
          }
      )
