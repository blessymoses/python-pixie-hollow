**********
cancel_run
**********



.. py:method:: Omics.Client.cancel_run(**kwargs)

  

  Cancels a run using its ID and returns a response with no body if the operation is successful. To confirm that the run has been cancelled, use the ``ListRuns`` API operation to check that it is no longer listed.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/omics-2022-11-28/CancelRun>`_  


  **Request Syntax**
  ::

    response = client.cancel_run(
        id='string'
    )
    
  :type id: string
  :param id: **[REQUIRED]** 

    The run's ID.

    

  
  
  :returns: None
