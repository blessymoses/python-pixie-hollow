************
delete_share
************



.. py:method:: Omics.Client.delete_share(**kwargs)

  

  Deletes a resource share. If you are the resource owner, the subscriber will no longer have access to the shared resource. If you are the subscriber, this operation deletes your access to the share.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/omics-2022-11-28/DeleteShare>`_  


  **Request Syntax**
  ::

    response = client.delete_share(
        shareId='string'
    )
    
  :type shareId: string
  :param shareId: **[REQUIRED]** 

    The ID for the resource share to be deleted.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'status': 'PENDING'|'ACTIVATING'|'ACTIVE'|'DELETING'|'DELETED'|'FAILED'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **status** *(string) --* 

        The status of the share being deleted.
