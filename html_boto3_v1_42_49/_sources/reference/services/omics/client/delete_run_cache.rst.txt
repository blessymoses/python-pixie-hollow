****************
delete_run_cache
****************



.. py:method:: Omics.Client.delete_run_cache(**kwargs)

  

  Deletes a run cache and returns a response with no body if the operation is successful. This action removes the cache metadata stored in the service account, but does not delete the data in Amazon S3. You can access the cache data in Amazon S3, for inspection or to troubleshoot issues. You can remove old cache data using standard S3 ``Delete`` operations.

   

  For more information, see `Deleting a run cache <https://docs.aws.amazon.com/omics/latest/dev/workflow-cache-delete.html>`__ in the *Amazon Web Services HealthOmics User Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/omics-2022-11-28/DeleteRunCache>`_  


  **Request Syntax**
  ::

    response = client.delete_run_cache(
        id='string'
    )
    
  :type id: string
  :param id: **[REQUIRED]** 

    Run cache identifier for the cache you want to delete.

    

  
  
  :returns: None
