***********
get_subject
***********



.. py:method:: IAMRolesAnywhere.Client.get_subject(**kwargs)

  

  Gets a *subject*, which associates a certificate identity with authentication attempts. The subject stores auditing information such as the status of the last authentication attempt, the certificate data used in the attempt, and the last time the associated identity attempted authentication.

   

  **Required permissions:** ``rolesanywhere:GetSubject``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rolesanywhere-2018-05-10/GetSubject>`_  


  **Request Syntax**
  ::

    response = client.get_subject(
        subjectId='string'
    )
    
  :type subjectId: string
  :param subjectId: **[REQUIRED]** 

    The unique identifier of the subject.

    

  
  
  :rtype: dict
  :returns:
