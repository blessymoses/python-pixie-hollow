list_crls

.. py:method:: IAMRolesAnywhere.Client.list_crls(**kwargs)

  **Required permissions:** ``rolesanywhere:ListCrls``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rolesanywhere-2018-05-10/ListCrls>`_
