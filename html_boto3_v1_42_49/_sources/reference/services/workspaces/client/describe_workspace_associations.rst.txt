.. py:method:: WorkSpaces.Client.describe_workspace_associations(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/workspaces-2015-04-08/DescribeWorkspaceAssociations>`_
