LexModelsV2

======
Client
======



.. py:class:: LexModelsV2.Client

  A low-level client representing Amazon Lex Model Building V2 (Lex Models V2)
  

  Amazon Lex Model Building Service provides APIs for creating, managing, and deploying conversational bots and their components.

  ::

    
    import boto3
    
    client = boto3.client('lexv2-models')
