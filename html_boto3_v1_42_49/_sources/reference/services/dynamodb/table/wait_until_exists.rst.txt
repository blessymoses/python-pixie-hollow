wait_until_exists
*****************



.. py:method:: DynamoDB.Table.wait_until_exists()

  Waits until this Table is exists. This method calls :py:meth:`DynamoDB.Waiter.table_exists.wait` which polls :py:meth:`DynamoDB.Client.describe_table` every 20 seconds until a successful state is reached. An error is raised after 25 failed checks.

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/dynamodb-2012-08-10/DescribeTable>`_  


  **Request Syntax**
  ::

    table.wait_until_exists()
    
    
  :returns: None
