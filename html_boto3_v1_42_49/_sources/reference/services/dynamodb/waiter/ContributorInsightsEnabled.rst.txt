  .. py:method:: wait(**kwargs)

    Polls :py:meth:`DynamoDB.Client.describe_contributor_insights` every 20 seconds until a successful state is reached. An error is raised after 30 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/dynamodb-2012-08-10/DescribeContributorInsights>`_    


    **Request Syntax**
    ::

      waiter.wait(
          TableName='string',
          IndexName='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type TableName: string
    :param TableName: **[REQUIRED]** 

      The name of the table to describe. You can also provide the Amazon Resource Name (ARN) of the table in this parameter.

      

    
    :type IndexName: string
    :param IndexName: 

      The name of the global secondary index to describe, if applicable.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 20

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 30

        

      
    
    
    :returns: None
