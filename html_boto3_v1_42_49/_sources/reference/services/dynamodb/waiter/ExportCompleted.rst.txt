  .. py:method:: wait(**kwargs)

    Polls :py:meth:`DynamoDB.Client.describe_export` every 20 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/dynamodb-2012-08-10/DescribeExport>`_    


    **Request Syntax**
    ::

      waiter.wait(
          ExportArn='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type ExportArn: string
    :param ExportArn: **[REQUIRED]** 

      The Amazon Resource Name (ARN) associated with the export.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 20

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60

        

      
    
    
    :returns: None
