Client
======



.. py:class:: ConnectCampaignService.Client

  A low-level client representing AmazonConnectCampaignService
  

  Provide APIs to create and manage Amazon Connect Campaigns.

  ::

    
    import boto3
    
    client = boto3.client('connectcampaigns')
