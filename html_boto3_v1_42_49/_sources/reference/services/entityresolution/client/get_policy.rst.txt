get_policy
**********



.. py:method:: EntityResolution.Client.get_policy(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/entityresolution-2018-05-10/GetPolicy>`_  


  **Request Syntax**
  ::

    response = client.get_policy(
        arn='string'
    )
    
  :type arn: string

  :returns:
