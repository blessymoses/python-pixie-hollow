get_configuration_set
*********************



.. py:method:: SESV2.Client.get_configuration_set(**kwargs)

  

  Get information about an existing configuration set, including the dedicated IP pool that it's associated with, whether or not it's enabled for sending email, and more.

   

  *Configuration sets* are groups of rules that you can apply to the emails you send. You apply a configuration set to an email by including a reference to the configuration set in the headers of the email. When you apply a configuration set to an email, all of the rules in that configuration set are applied to the email.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/sesv2-2019-09-27/GetConfigurationSet>`_  


  **Request Syntax**
  ::

    response = client.get_configuration_set(
        ConfigurationSetName='string'
    )
    
  :type ConfigurationSetName: string
  :param ConfigurationSetName: **[REQUIRED]** 

    The name of the configuration set.
