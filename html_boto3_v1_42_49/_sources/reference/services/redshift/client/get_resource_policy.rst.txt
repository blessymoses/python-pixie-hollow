get_resource_policy
*******************



.. py:method:: Redshift.Client.get_resource_policy(**kwargs)

  

  Get the resource policy for a specified resource.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/redshift-2012-12-01/GetResourcePolicy>`_  


  **Request Syntax**
  ::

    response = client.get_resource_policy(
        ResourceArn='string'
    )
    
  :type ResourceArn: string
  :param ResourceArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the resource of which its resource policy is fetched.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'ResourcePolicy': {
              'ResourceArn': 'string',
              'Policy': 'string'
          }
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **ResourcePolicy** *(dict) --* 

        The content of the resource policy.

        
        

        - **ResourceArn** *(string) --* 

          The resources that a policy is attached to.

          
        

        - **Policy** *(string) --* 

          The content of a resource policy.

          
    
  
  **Exceptions**
  
  *   :py:class:`Redshift.Client.exceptions.ResourceNotFoundFault`

  
  *   :py:class:`Redshift.Client.exceptions.InvalidPolicyFault`
