.. py:method:: Redshift.Client.rotate_encryption_key(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/redshift-2012-12-01/RotateEncryptionKey>`_

  :type ClusterIdentifier: string
