create_hsm_client_certificate
*****************************



.. py:method:: Redshift.Client.create_hsm_client_certificate(**kwargs)

  

  Creates an HSM client certificate that an Amazon Redshift cluster will use to connect to the client's HSM in order to store and retrieve the keys used to encrypt the cluster databases.

   

  The command returns a public key, which you must store in the HSM. In addition to creating the HSM certificate, you must create an Amazon Redshift HSM configuration that provides a cluster the information needed to store and use encryption keys in the HSM. For more information, go to `Hardware Security Modules <https://docs.aws.amazon.com/redshift/latest/mgmt/working-with-db-encryption.html#working-with-HSM>`__ in the *Amazon Redshift Cluster Management Guide*.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/redshift-2012-12-01/CreateHsmClientCertificate>`_  


  **Request Syntax**

    response = client.create_hsm_client_certificate(

        ]
    )
    
  :type HsmClientCertificateIdentifier: string
  :param HsmClientCertificateIdentifier: **[REQUIRED]** 

    The identifier to be assigned to the new HSM client certificate that the cluster will use to connect to the HSM to use the database encryption keys.
