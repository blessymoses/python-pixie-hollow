.. py:method:: MediaTailor.Client.put_channel_policy(**kwargs)

        ChannelName='string',
        Policy='string'
    )
    
  :type ChannelName: string
  :param ChannelName: **[REQUIRED]** 

    The channel name associated with this Channel Policy.

    

  
  :type Policy: string
  :param Policy: **[REQUIRED]**
