delete_pricing_plan
*******************



.. py:method:: BillingConductor.Client.delete_pricing_plan(**kwargs)

  

  Deletes a pricing plan. The pricing plan must not be associated with any billing groups to delete successfully.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/billingconductor-2021-07-30/DeletePricingPlan>`_  


  **Request Syntax**
  ::

    response = client.delete_pricing_plan(
        Arn='string'
    )
    
  :type Arn: string
  :param Arn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the pricing plan that you're deleting.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'Arn': 'string'
      }
      
    **Response Structure**

    

    - *(dict) --* 
      

      - **Arn** *(string) --* 

        The Amazon Resource Name (ARN) of the deleted pricing plan.
