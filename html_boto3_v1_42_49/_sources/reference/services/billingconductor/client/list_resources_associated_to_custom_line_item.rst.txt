  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/billingconductor-2021-07-30/ListResourcesAssociatedToCustomLineItem>`_  


  **Request Syntax**

    response = client.list_resources_associated_to_custom_line_item(

        NextToken='string',

  :param BillingPeriod:
