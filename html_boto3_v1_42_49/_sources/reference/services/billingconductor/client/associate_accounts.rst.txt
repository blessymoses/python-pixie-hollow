******************
associate_accounts
******************



.. py:method:: BillingConductor.Client.associate_accounts(**kwargs)

  

  Connects an array of account IDs in a consolidated billing family to a predefined billing group. The account IDs must be a part of the consolidated billing family during the current month, and not already associated with another billing group. The maximum number of accounts that can be associated in one call is 30.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/billingconductor-2021-07-30/AssociateAccounts>`_  


  **Request Syntax**
  ::

    response = client.associate_accounts(
        Arn='string',
        AccountIds=[
            'string',
        ]
    )
    
  :type Arn: string
  :param Arn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the billing group that associates the array of account IDs.

    

  
  :type AccountIds: list
  :param AccountIds: **[REQUIRED]** 

    The associating array of account IDs.
