======
Client
======



.. py:class:: Inspector.Client

  A low-level client representing Amazon Inspector
   

  Amazon Inspector enables you to analyze the behavior of your AWS resources and to identify potential security issues. For more information, see `Amazon Inspector User Guide <https://docs.aws.amazon.com/inspector/latest/userguide/inspector_introduction.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('inspector')
