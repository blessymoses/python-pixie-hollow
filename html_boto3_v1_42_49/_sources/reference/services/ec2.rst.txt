======
Client
======



.. py:class:: EC2.Client

  A low-level client representing Amazon Elastic Compute Cloud (EC2)
   

  You can access the features of Amazon Elastic Compute Cloud (Amazon EC2) programmatically. For more information, see the `Amazon EC2 Developer Guide <https://docs.aws.amazon.com/ec2/latest/devguide>`__.

  ::

    
    import boto3
    
    client = boto3.client('ec2')
