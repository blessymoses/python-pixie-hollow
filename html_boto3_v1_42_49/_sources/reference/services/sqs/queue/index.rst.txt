=====
Queue
=====



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: SQS.Queue(url)

  A resource representing an Amazon Simple Queue Service (SQS) Queue::

    
    import boto3
    
    sqs = boto3.resource('sqs')
    queue = sqs.Queue('url')

  :type url: string
  :param url: The Queue's url identifier. This **must** be set.
