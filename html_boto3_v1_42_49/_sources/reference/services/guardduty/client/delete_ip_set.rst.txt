*************
delete_ip_set
*************



.. py:method:: GuardDuty.Client.delete_ip_set(**kwargs)

  

  Deletes the IPSet specified by the ``ipSetId``. IPSets are called trusted IP lists in the console user interface.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/guardduty-2017-11-28/DeleteIPSet>`_  


  **Request Syntax**
  ::

    response = client.delete_ip_set(
        DetectorId='string',
        IpSetId='string'
    )
    
  :type DetectorId: string
  :param DetectorId: **[REQUIRED]** 

    The unique ID of the detector associated with the IPSet.

     

    To find the ``detectorId`` in the current Region, see the Settings page in the GuardDuty console, or run the `ListDetectors <https://docs.aws.amazon.com/guardduty/latest/APIReference/API_ListDetectors.html>`__ API.

    

  
  :type IpSetId: string
  :param IpSetId: **[REQUIRED]** 

    The unique ID of the IPSet to delete.
