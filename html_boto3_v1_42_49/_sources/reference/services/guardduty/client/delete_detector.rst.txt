delete_detector
***************



.. py:method:: GuardDuty.Client.delete_detector(**kwargs)

  

  Deletes an Amazon GuardDuty detector that is specified by the detector ID.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/guardduty-2017-11-28/DeleteDetector>`_  


  **Request Syntax**
  ::

    response = client.delete_detector(
        DetectorId='string'
    )
    
  :type DetectorId: string
  :param DetectorId: **[REQUIRED]** 

    The unique ID of the detector that you want to delete.

     

    To find the ``detectorId`` in the current Region, see the Settings page in the GuardDuty console, or run the `ListDetectors <https://docs.aws.amazon.com/guardduty/latest/APIReference/API_ListDetectors.html>`__ API.
