stop_search_job
***************



.. py:method:: BackupSearch.Client.stop_search_job(**kwargs)

  

  This operations ends a search job.

   

  Only a search job with a status of ``RUNNING`` can be stopped.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/backupsearch-2018-05-10/StopSearchJob>`_  


  **Request Syntax**
  ::

    response = client.stop_search_job(
        SearchJobIdentifier='string'
    )
    
  :type SearchJobIdentifier: string
  :param SearchJobIdentifier: **[REQUIRED]** 

    The unique string that specifies the search job.
