list_devices_for_wireless_device_import_task

.. py:method:: IoTWireless.Client.list_devices_for_wireless_device_import_task(**kwargs)

  

  List the Sidewalk devices in an import task and their onboarding status.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iotwireless-2020-11-22/ListDevicesForWirelessDeviceImportTask>`_  


  **Request Syntax**
  ::

    response = client.list_devices_for_wireless_device_import_task(
        Id='string',
        MaxResults=123,
        NextToken='string',
        Status='INITIALIZED'|'PENDING'|'ONBOARDED'|'FAILED'
    )
    
  :type Id: string
  :param Id: **[REQUIRED]** 

    The identifier of the import task for which wireless devices are listed.

    

  
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of results to return in this operation.

    

  
  :type NextToken: string
  :param NextToken: 

    To retrieve the next set of results, the ``nextToken`` value from a previous response; otherwise ``null`` to receive the first set of results.

    

  
  :type Status: string
  :param Status: 

    The status of the devices in the import task.

    

  
  
  :rtype: dict
  :returns:
