list_identity_propagation_configs

.. py:method:: QuickSight.Client.list_identity_propagation_configs(**kwargs)

  

  Lists all services and authorized targets that the Quick Sight IAM Identity Center application can access.

   

  This operation is only supported for Quick Sight accounts that use IAM Identity Center.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/ListIdentityPropagationConfigs>`_
