list_topics
***********



.. py:method:: QuickSight.Client.list_topics(**kwargs)

  

  Lists all of the topics within an account.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/ListTopics>`_  


  **Request Syntax**
  ::

    response = client.list_topics(
        AwsAccountId='string',
        NextToken='string',
        MaxResults=123
    )
    
  :type AwsAccountId: string
  :param AwsAccountId: **[REQUIRED]** 

    The ID of the Amazon Web Services account that contains the topics that you want to list.

    

  
  :type NextToken: string
  :param NextToken: 

    The token for the next set of results, or null if there are no more results.

    

  
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of results to be returned per request.
