describe_topic_permissions
**************************



.. py:method:: QuickSight.Client.describe_topic_permissions(**kwargs)

  

  Describes the permissions of a topic.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/DescribeTopicPermissions>`_  


  **Request Syntax**
  ::

    response = client.describe_topic_permissions(
        AwsAccountId='string',
        TopicId='string'
    )
    
  :type AwsAccountId: string
  :param AwsAccountId: **[REQUIRED]** 

    The ID of the Amazon Web Services account that contains the topic that you want described.

    

  
  :type TopicId: string
  :param TopicId: **[REQUIRED]** 

    The ID of the topic that you want to describe. This ID is unique per Amazon Web Services Region for each Amazon Web Services account.

    

  
  
  :rtype: dict
  :returns:

Use this option only to share resources (templates) across Amazon Web Services accounts. Account root sharing is not supported for action connectors.
