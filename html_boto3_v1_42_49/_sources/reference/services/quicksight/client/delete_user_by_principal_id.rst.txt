.. py:method:: QuickSight.Client.delete_user_by_principal_id(**kwargs)

  

  Deletes a user identified by its principal ID.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/DeleteUserByPrincipalId>`_  


  **Request Syntax**
  ::

    response = client.delete_user_by_principal_id(
        PrincipalId='string',
        AwsAccountId='string',
        Namespace='string'
    )
    
  :type PrincipalId: string
  :param PrincipalId: **[REQUIRED]** 

    The principal ID of the user.

    

  
  :type AwsAccountId: string
  :param AwsAccountId: **[REQUIRED]** 

    The ID for the Amazon Web Services account that the user is in. Currently, you use the ID for the Amazon Web Services account that contains your Amazon Quick Sight account.
