list_folders
************



.. py:method:: QuickSight.Client.list_folders(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/ListFolders>`_  


  **Request Syntax**
  ::

    response = client.list_folders(
        AwsAccountId='string',
        NextToken='string',
        MaxResults=123
    )
    
  :type AwsAccountId: string
  :param AwsAccountId: **[REQUIRED]**
