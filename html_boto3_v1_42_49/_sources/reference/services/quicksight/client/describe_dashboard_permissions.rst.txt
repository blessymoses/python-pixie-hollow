describe_dashboard_permissions
******************************



.. py:method:: QuickSight.Client.describe_dashboard_permissions(**kwargs)

  

  Describes read and write permissions for a dashboard.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/DescribeDashboardPermissions>`_  


  **Request Syntax**
  ::

    response = client.describe_dashboard_permissions(
        AwsAccountId='string',
        DashboardId='string'
    )
    
  :type AwsAccountId: string
  :param AwsAccountId: **[REQUIRED]** 

    The ID of the Amazon Web Services account that contains the dashboard that you're describing permissions for.

    

  
  :type DashboardId: string
  :param DashboardId: **[REQUIRED]** 

    The ID for the dashboard, also added to the IAM policy.

    

  
  
  :rtype: dict
  :returns:
