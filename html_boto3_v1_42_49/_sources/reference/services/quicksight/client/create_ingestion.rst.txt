****************
create_ingestion
****************



.. py:method:: QuickSight.Client.create_ingestion(**kwargs)

  

  Creates and starts a new SPICE ingestion for a dataset. You can manually refresh datasets in an Enterprise edition account 32 times in a 24-hour period. You can manually refresh datasets in a Standard edition account 8 times in a 24-hour period. Each 24-hour period is measured starting 24 hours before the current date and time.

   

  Any ingestions operating on tagged datasets inherit the same tags automatically for use in access control. For an example, see `How do I create an IAM policy to control access to Amazon EC2 resources using tags? <http://aws.amazon.com/premiumsupport/knowledge-center/iam-ec2-resource-tags/>`__ in the Amazon Web Services Knowledge Center. Tags are visible on the tagged dataset, but not on the ingestion resource.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/quicksight-2018-04-01/CreateIngestion>`_  


  **Request Syntax**

    response = client.create_ingestion(
        DataSetId='string',
        IngestionId='string',
        AwsAccountId='string',
        IngestionType='INCREMENTAL_REFRESH'|'FULL_REFRESH'
    )
