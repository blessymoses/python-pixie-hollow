list_kx_dataviews
*****************



.. py:method:: finspace.Client.list_kx_dataviews(**kwargs)

  

  Returns a list of all the dataviews in the database.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/finspace-2021-03-12/ListKxDataviews>`_  


  **Request Syntax**
  ::

    response = client.list_kx_dataviews(
        environmentId='string',
        databaseName='string',
        nextToken='string',
        maxResults=123
    )
    
  :type environmentId: string
  :param environmentId: **[REQUIRED]** 

    A unique identifier for the kdb environment, for which you want to retrieve a list of dataviews.

    

  
  :type databaseName: string
  :param databaseName: **[REQUIRED]** 

    The name of the database where the dataviews were created.

    

  
  :type nextToken: string
  :param nextToken: 

    A token that indicates where a results page should begin.

    

  
  :type maxResults: integer
