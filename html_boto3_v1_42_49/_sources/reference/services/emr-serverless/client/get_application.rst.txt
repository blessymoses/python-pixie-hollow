get_application
***************



.. py:method:: EMRServerless.Client.get_application(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/emr-serverless-2021-07-13/GetApplication>`_

  ::

        applicationId='string'
