***************************
update_launch_configuration
***************************



.. py:method:: mgn.Client.update_launch_configuration(**kwargs)

  

  Updates multiple LaunchConfigurations by Source Server ID.

   

  .. note::

    

    bootMode valid values are ``LEGACY_BIOS | UEFI``

    

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mgn-2020-02-26/UpdateLaunchConfiguration>`_  


  **Request Syntax**
