disassociate_user

.. py:method:: LicenseManagerUserSubscriptions.Client.disassociate_user(**kwargs)

  

  Disassociates the user from an EC2 instance providing user-based subscriptions.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/license-manager-user-subscriptions-2018-05-10/DisassociateUser>`_
