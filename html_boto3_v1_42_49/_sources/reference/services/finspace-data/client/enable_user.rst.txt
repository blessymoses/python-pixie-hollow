***********
enable_user
***********



.. py:method:: FinSpaceData.Client.enable_user(**kwargs)

  

  Allows the specified user to access the FinSpace web application and API.

  

  .. danger::

        This operation is deprecated and may not function as expected. This operation should not be used going forward and is only kept for the purpose of backwards compatiblity.


  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/finspace-2020-07-13/EnableUser>`_  


  **Request Syntax**
  ::

    response = client.enable_user(
        userId='string',
        clientToken='string'
    )
    
  :type userId: string
  :param userId: **[REQUIRED]** 

    The unique identifier for the user that you want to activate.

    

  
  :type clientToken: string
  :param clientToken: 

    A token that ensures idempotency. This token expires in 10 minutes.

    This field is autopopulated if not provided.

  :returns: 
    
    **Response Syntax**

          'userId': 'string'
      }
