get_protocols_list

.. py:method:: FMS.Client.get_protocols_list(**kwargs)

  

  Returns information about the specified Firewall Manager protocols list.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/fms-2018-01-01/GetProtocolsList>`_  


  **Request Syntax**
  ::

    response = client.get_protocols_list(
        ListId='string',
        DefaultList=True|False
    )
    
  :type ListId: string
  :param ListId: **[REQUIRED]** 

    The ID of the Firewall Manager protocols list that you want the details for.

    

  
  :type DefaultList: boolean
  :param DefaultList: 

    Specifies whether the list to retrieve is a default list owned by Firewall Manager.
