delete_apps_list
****************



.. py:method:: FMS.Client.delete_apps_list(**kwargs)

  

  Permanently deletes an Firewall Manager applications list.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/fms-2018-01-01/DeleteAppsList>`_  


  **Request Syntax**
  ::

    response = client.delete_apps_list(
        ListId='string'
    )
    
  :type ListId: string
  :param ListId: **[REQUIRED]** 

    The ID of the applications list that you want to delete. You can retrieve this ID from ``PutAppsList``, ``ListAppsLists``, and ``GetAppsList``.

    

  
  
  :returns: None
