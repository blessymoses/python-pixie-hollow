**********
cancel_job
**********



.. py:method:: DataExchange.Client.cancel_job(**kwargs)

  

  This operation cancels a job. Jobs can be cancelled only when they are in the WAITING state.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/dataexchange-2017-07-25/CancelJob>`_  


  **Request Syntax**
  ::

    response = client.cancel_job(
        JobId='string'
    )
    
  :type JobId: string
  :param JobId: **[REQUIRED]** 

    The unique identifier for a job.

    

  
  
  :returns: None
