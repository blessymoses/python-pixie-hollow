***********************************
register_marketplace_model_endpoint
***********************************



.. py:method:: Bedrock.Client.register_marketplace_model_endpoint(**kwargs)

  

  Registers an existing Amazon SageMaker endpoint with Amazon Bedrock Marketplace, allowing it to be used with Amazon Bedrock APIs.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/bedrock-2023-04-20/RegisterMarketplaceModelEndpoint>`_  


  **Request Syntax**
  ::

    response = client.register_marketplace_model_endpoint(
        endpointIdentifier='string',
        modelSourceIdentifier='string'
    )
    
  :type endpointIdentifier: string
  :param endpointIdentifier: **[REQUIRED]** 

    The ARN of the Amazon SageMaker endpoint you want to register with Amazon Bedrock Marketplace.

    

  
  :type modelSourceIdentifier: string
  :param modelSourceIdentifier: **[REQUIRED]** 

    The ARN of the model from Amazon Bedrock Marketplace that is deployed on the endpoint.

    

  
  
  :rtype: dict
  :returns:
