======
Client
======



.. py:class:: FinSpaceData.Client

  A low-level client representing FinSpace Public API (FinSpace Data)
  

  The FinSpace APIs let you take actions inside the FinSpace.

  ::

    
    import boto3
    
    client = boto3.client('finspace-data')
