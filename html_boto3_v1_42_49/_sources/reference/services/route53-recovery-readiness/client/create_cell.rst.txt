.. py:method:: Route53RecoveryReadiness.Client.create_cell(**kwargs)

  

  Creates a cell in an account.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/route53-recovery-readiness-2019-12-02/CreateCell>`_  


  **Request Syntax**
  ::

    response = client.create_cell(
        CellName='string',
        Cells=[
            'string',
        ],
        Tags={
            'string': 'string'
        }
    )
    
  :type CellName: string
  :param CellName: **[REQUIRED]**
