*****************
delete_public_key
*****************



.. py:method:: CloudFront.Client.delete_public_key(**kwargs)

  

  Remove a public key you previously added to CloudFront.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudfront-2020-05-31/DeletePublicKey>`_  


  **Request Syntax**
  ::

    response = client.delete_public_key(
        Id='string',
        IfMatch='string'
    )
    
  :type Id: string
  :param Id: **[REQUIRED]** 

    The ID of the public key you want to remove from CloudFront.

    

  
  :type IfMatch: string
  :param IfMatch: 

    The value of the ``ETag`` header that you received when retrieving the public key identity to delete. For example: ``E2QWRUHAPOMQZL``.

    

  
  
  :returns: None
  **Exceptions**
