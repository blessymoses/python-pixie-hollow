delete_vpc_origin
*****************



.. py:method:: CloudFront.Client.delete_vpc_origin(**kwargs)

  

  Delete an Amazon CloudFront VPC origin.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudfront-2020-05-31/DeleteVpcOrigin>`_  


  **Request Syntax**
  ::

    response = client.delete_vpc_origin(
        Id='string',
        IfMatch='string'
    )
    
  :type Id: string
  :param Id: **[REQUIRED]** 

    The VPC origin ID.

    

  
  :type IfMatch: string
  :param IfMatch: **[REQUIRED]** 

    The version identifier of the VPC origin to delete. This is the ``ETag`` value returned in the response to  GetVpcOrigin.

    

  
  
  :rtype: dict
  :returns:
