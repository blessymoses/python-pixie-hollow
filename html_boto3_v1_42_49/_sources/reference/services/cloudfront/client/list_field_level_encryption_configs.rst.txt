list_field_level_encryption_configs
***********************************



.. py:method:: CloudFront.Client.list_field_level_encryption_configs(**kwargs)

  

  List all field-level encryption configurations that have been created in CloudFront for this account.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudfront-2020-05-31/ListFieldLevelEncryptionConfigs>`_  


  **Request Syntax**
  ::

    response = client.list_field_level_encryption_configs(
        Marker='string',
        MaxItems='string'
    )
    
  :type Marker: string
  :param Marker: 

    Use this when paginating results to indicate where to begin in your list of configurations. The results include configurations in the list that occur after the marker. To get the next page of results, set the ``Marker`` to the value of the ``NextMarker`` from the current page's response (which is also the ID of the last configuration on that page).

    

  
  :type MaxItems: string
  :param MaxItems: 

    The maximum number of field-level encryption configurations you want in the response body.

    

  
  
  :rtype: dict
  :returns:
