  .. py:method:: wait(**kwargs)

    Polls :py:meth:`CloudFront.Client.get_invalidation_for_distribution_tenant` every 20 seconds until a successful state is reached. An error is raised after 30 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudfront-2020-05-31/GetInvalidationForDistributionTenant>`_    


    **Request Syntax**
    ::

      waiter.wait(
          DistributionTenantId='string',
          Id='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type DistributionTenantId: string
    :param DistributionTenantId: **[REQUIRED]** 

      The ID of the distribution tenant.

      

    
    :type Id: string
    :param Id: **[REQUIRED]** 

      The ID of the invalidation to retrieve.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 20

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 30

        

      
    
    
    :returns: None
