  .. py:method:: wait(**kwargs)

    Polls :py:meth:`CloudFront.Client.get_distribution` every 60 seconds until a successful state is reached. An error is raised after 35 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/cloudfront-2020-05-31/GetDistribution>`_    


    **Request Syntax**
    ::

      waiter.wait(
          Id='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type Id: string
    :param Id: **[REQUIRED]** 

      The distribution's ID. If the ID is empty, an empty distribution configuration is returned.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 60

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 35

        

      
    
    
    :returns: None
