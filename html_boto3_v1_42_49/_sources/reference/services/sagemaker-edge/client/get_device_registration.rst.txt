.. py:method:: SagemakerEdgeManager.Client.get_device_registration(**kwargs)

  **Request Syntax**

        DeviceName='string',
        DeviceFleetName='string'
    )
    
  :type DeviceName: string
  :param DeviceName: **[REQUIRED]**
