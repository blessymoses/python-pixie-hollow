create_core_network
*******************



.. py:method:: NetworkManager.Client.create_core_network(**kwargs)

  

  Creates a core network as part of your global network, and optionally, with a core network policy.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/networkmanager-2019-07-05/CreateCoreNetwork>`_  


  **Request Syntax**
  ::

    response = client.create_core_network(
        GlobalNetworkId='string',
        Description='string',
        Tags=[
            {
                'Key': 'string',
                'Value': 'string'
            },
        ],
        PolicyDocument='string',
        ClientToken='string'
    )
    
  :type GlobalNetworkId: string
  :param GlobalNetworkId: **[REQUIRED]**

  :type Description: string
