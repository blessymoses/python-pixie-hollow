.. py:method:: NetworkManager.Client.put_attachment_routing_policy_label(**kwargs)

  **Request Syntax**

        AttachmentId='string',
        RoutingPolicyLabel='string',
        ClientToken='string'
    )

  :param CoreNetworkId: **[REQUIRED]**

  :type AttachmentId: string
