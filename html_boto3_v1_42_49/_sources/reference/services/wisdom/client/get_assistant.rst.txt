get_assistant

.. py:method:: ConnectWisdomService.Client.get_assistant(**kwargs)

  

  Retrieves information about an assistant.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/wisdom-2020-10-19/GetAssistant>`_  


  **Request Syntax**
  ::

    response = client.get_assistant(
        assistantId='string'
    )
    
  :type assistantId: string
  :param assistantId: **[REQUIRED]** 

    The identifier of the Wisdom assistant. Can be either the ID or the ARN. URLs cannot contain the ARN.
