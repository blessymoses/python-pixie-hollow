get_import_job
**************



.. py:method:: ConnectWisdomService.Client.get_import_job(**kwargs)

  

  Retrieves the started import job.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/wisdom-2020-10-19/GetImportJob>`_  


  **Request Syntax**
  ::

    response = client.get_import_job(
        importJobId='string',
        knowledgeBaseId='string'
    )
    
  :type importJobId: string
  :param importJobId: **[REQUIRED]** 

    The identifier of the import job to retrieve.

    

  
  :type knowledgeBaseId: string
  :param knowledgeBaseId: **[REQUIRED]** 

    The identifier of the knowledge base that the import job belongs to.

    

  
  
  :rtype: dict
  :returns:
