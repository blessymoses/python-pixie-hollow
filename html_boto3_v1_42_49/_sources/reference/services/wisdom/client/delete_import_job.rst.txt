  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/wisdom-2020-10-19/DeleteImportJob>`_  


  **Request Syntax**

    response = client.delete_import_job(
        importJobId='string',
        knowledgeBaseId='string'
    )
    
  :type importJobId: string
  :param importJobId: **[REQUIRED]** 

    The identifier of the import job to be deleted.

    

  
  :type knowledgeBaseId: string
  :param knowledgeBaseId: **[REQUIRED]** 

    The identifier of the knowledge base. This should not be a QUICK_RESPONSES type knowledge base if you're storing Wisdom Content resource to it.
