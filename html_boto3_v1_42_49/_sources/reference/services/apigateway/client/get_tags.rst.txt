get_tags

.. py:method:: APIGateway.Client.get_tags(**kwargs)

  **Request Syntax**

    response = client.get_tags(
        resourceArn='string',
        position='string',
        limit=123
    )
    
  :type resourceArn: string

    The ARN of a resource that can be tagged.
