.. py:method:: APIGateway.Client.get_vpc_links(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/apigateway-2015-07-09/GetVpcLinks>`_  


  **Request Syntax**

        limit=123

  :type limit: integer
  :param limit:
