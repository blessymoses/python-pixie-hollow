======
Client
======



.. py:class:: AgentsforBedrock.Client

  A low-level client representing Agents for Amazon Bedrock
  

  Describes the API operations for creating and managing Amazon Bedrock agents.

  ::

    
    import boto3
    
    client = boto3.client('bedrock-agent')
