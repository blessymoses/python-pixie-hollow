put_profile_visibility
**********************



.. py:method:: PartnerCentralAccountAPI.Client.put_profile_visibility(**kwargs)

  

  Sets the visibility level for a partner profile, controlling who can view the profile information.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/partnercentral-account-2025-04-04/PutProfileVisibility>`_  


  **Request Syntax**
  ::

    response = client.put_profile_visibility(
        Catalog='string',
        Identifier='string',
        Visibility='PRIVATE'|'PUBLIC'
    )
    
  :type Catalog: string
  :param Catalog: **[REQUIRED]** 

    The catalog identifier for the partner account.

    

  
  :type Identifier: string
  :param Identifier: **[REQUIRED]** 

    The unique identifier of the partner account.

    

  
  :type Visibility: string
  :param Visibility: **[REQUIRED]** 

    The visibility setting to apply to the partner profile.

    

  
  
  :rtype: dict
  :returns: 
    
    **Response Syntax**

    
    ::

      {
          'Catalog': 'string',
          'Arn': 'string',
          'Id': 'string',
          'Visibility': 'PRIVATE'|'PUBLIC',
          'ProfileId': 'string'
      }
