put_configuration
*****************



.. py:method:: AppRegistry.Client.put_configuration(**kwargs)

  

  Associates a ``TagKey`` configuration to an account.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/AWS242AppRegistry-2020-06-24/PutConfiguration>`_  


  **Request Syntax**
  ::

    response = client.put_configuration(
        configuration={
            'tagQueryConfiguration': {
                'tagKey': 'string'
            }
        }
    )
    
  :type configuration: dict
  :param configuration: **[REQUIRED]** 

    Associates a ``TagKey`` configuration to an account.

    

  
    - **tagQueryConfiguration** *(dict) --* 

      Includes the definition of a ``tagQuery``.

      

    
      - **tagKey** *(string) --* 

        Condition in the IAM policy that associates resources to an application.

        

      
    
  
  
  :returns: None
