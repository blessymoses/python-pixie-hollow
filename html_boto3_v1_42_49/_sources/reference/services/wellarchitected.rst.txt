======
Client
======



.. py:class:: WellArchitected.Client

  A low-level client representing AWS Well-Architected Tool
   

  This is the *Well-Architected Tool API Reference*. The WA Tool API provides programmatic access to the `Well-Architected Tool <http://aws.amazon.com/well-architected-tool>`__ in the `Amazon Web Services Management Console <https://console.aws.amazon.com/wellarchitected>`__. For information about the Well-Architected Tool, see the `Well-Architected Tool User Guide <https://docs.aws.amazon.com/wellarchitected/latest/userguide/intro.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('wellarchitected')
