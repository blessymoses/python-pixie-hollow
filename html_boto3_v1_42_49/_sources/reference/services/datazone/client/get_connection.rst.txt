:doc:`DataZone <../../datazone>` / Client / get_connection

**************
get_connection
**************



.. py:method:: DataZone.Client.get_connection(**kwargs)

  

  Gets a connection. In Amazon DataZone, a connection enables you to connect your resources (domains, projects, and environments) to external resources and services.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/datazone-2018-05-10/GetConnection>`_  


  **Request Syntax**
  ::

    response = client.get_connection(
        domainIdentifier='string',
        identifier='string',
        withSecret=True|False
    )
    
  :type domainIdentifier: string
  :param domainIdentifier: **[REQUIRED]** 

    The ID of the domain where we get the connection.

    

  
  :type identifier: string
  :param identifier: **[REQUIRED]** 

    The connection ID.

    

  
  :type withSecret: boolean
  :param withSecret: 

    Specifies whether a connection has a secret.

    

  
  
  :rtype: dict
