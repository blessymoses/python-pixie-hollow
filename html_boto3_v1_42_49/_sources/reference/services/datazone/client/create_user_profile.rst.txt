.. py:method:: DataZone.Client.create_user_profile(**kwargs)

  

  Creates a user profile in Amazon DataZone.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/datazone-2018-05-10/CreateUserProfile>`_  


  **Request Syntax**
  ::

    response = client.create_user_profile(
        domainIdentifier='string',
        userIdentifier='string',
        userType='IAM_USER'|'IAM_ROLE'|'SSO_USER',
        clientToken='string'
    )
    
  :type domainIdentifier: string
  :param domainIdentifier: **[REQUIRED]** 

    The identifier of the Amazon DataZone domain in which a user profile is created.

    

  
  :type userIdentifier: string
  :param userIdentifier: **[REQUIRED]** 

    The identifier of the user for which the user profile is created.

    

  
  :type userType: string
  :param userType: 

    The user type of the user for which the user profile is created.

    

  
  :type clientToken: string
  :param clientToken:
