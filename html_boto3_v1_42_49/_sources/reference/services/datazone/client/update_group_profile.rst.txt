.. py:method:: DataZone.Client.update_group_profile(**kwargs)

  

  Updates the specified group profile in Amazon DataZone.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/datazone-2018-05-10/UpdateGroupProfile>`_  


  **Request Syntax**
  ::

    response = client.update_group_profile(
        domainIdentifier='string',
        groupIdentifier='string',
        status='ASSIGNED'|'NOT_ASSIGNED'
    )
    
  :type domainIdentifier: string
  :param domainIdentifier: **[REQUIRED]** 

    The identifier of the Amazon DataZone domain in which a group profile is updated.

    

  
  :type groupIdentifier: string
  :param groupIdentifier: **[REQUIRED]** 

    The identifier of the group profile that is updated.

    

  
  :type status: string
  :param status: **[REQUIRED]**
