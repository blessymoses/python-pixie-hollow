.. py:method:: DataZone.Client.remove_entity_owner(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/datazone-2018-05-10/RemoveEntityOwner>`_  


  **Request Syntax**

    response = client.remove_entity_owner(

        entityType='DOMAIN_UNIT',
        entityIdentifier='string',
        owner={
            'user': {
                'userIdentifier': 'string'
            },
            'group': {
                'groupIdentifier': 'string'
            }
        },
        clientToken='string'
    )
    
  :type domainIdentifier: string
  :param domainIdentifier: **[REQUIRED]** 

    The ID of the domain where you want to remove an owner from an entity.

    

  
  :type entityType: string
  :param entityType: **[REQUIRED]** 

    The type of the entity from which you want to remove an owner.

    

  
  :type entityIdentifier: string
  :param entityIdentifier: **[REQUIRED]** 

    The ID of the entity from which you want to remove an owner.

    

  
  :type owner: dict
  :param owner: **[REQUIRED]**

      Specifies that the domain unit owner is a group.
