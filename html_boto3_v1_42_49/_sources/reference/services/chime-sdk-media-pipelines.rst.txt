======
Client
======



.. py:class:: ChimeSDKMediaPipelines.Client

  A low-level client representing Amazon Chime SDK Media Pipelines
  

  The Amazon Chime SDK media pipeline APIs in this section allow software developers to create Amazon Chime SDK media pipelines that capture, concatenate, or stream your Amazon Chime SDK meetings. For more information about media pipelines, see `Amazon Chime SDK media pipelines <https://docs.aws.amazon.com/chime-sdk/latest/APIReference/API_Operations_Amazon_Chime_SDK_Media_Pipelines.html>`__.

  ::

    
    import boto3
    
    client = boto3.client('chime-sdk-media-pipelines')
