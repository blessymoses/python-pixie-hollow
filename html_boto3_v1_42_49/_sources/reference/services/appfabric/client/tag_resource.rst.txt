tag_resource
************



.. py:method:: AppFabric.Client.tag_resource(**kwargs)

  

  Assigns one or more tags (key-value pairs) to the specified resource.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/appfabric-2023-05-19/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        resourceArn='string',
        tags=[
            {
                'key': 'string',
                'value': 'string'
            },
        ]
    )
    
  :type resourceArn: string
  :param resourceArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the resource that you want to tag.

    

  
  :type tags: list
  :param tags: **[REQUIRED]** 

    A map of the key-value pairs of the tag or tags to assign to the resource.

    

  
    - *(dict) --* 

      The key or keys of the key-value pairs for the tag or tags assigned to a resource.

      

    
      - **key** *(string) --* **[REQUIRED]** 

        Tag key.

        

      
      - **value** *(string) --* **[REQUIRED]** 

        Tag value.

        

      
    

  
  :rtype: dict
  :returns:

      {}
