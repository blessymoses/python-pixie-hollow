cancel_transaction
******************



.. py:method:: LakeFormation.Client.cancel_transaction(**kwargs)

  

  Attempts to cancel the specified transaction. Returns an exception if the transaction was previously committed.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/lakeformation-2017-03-31/CancelTransaction>`_

        TransactionId='string'

  :type TransactionId: string
  :param TransactionId: **[REQUIRED]**
