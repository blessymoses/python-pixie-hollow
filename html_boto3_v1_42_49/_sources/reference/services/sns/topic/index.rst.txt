Topic
=====



.. note::

  Before using anything on this page, please refer to the resources :doc:`user guide <../../../../guide/resources>` for the most recent guidance on using resources.



.. py:class:: SNS.Topic(arn)

  A resource representing an Amazon Simple Notification Service (SNS) Topic::

    
    import boto3
    
    sns = boto3.resource('sns')
    topic = sns.Topic('arn')

  :type arn: string
  :param arn: The Topic's arn identifier. This **must** be set.
