delete_extension_association
****************************



.. py:method:: AppConfig.Client.delete_extension_association(**kwargs)

  

  Deletes an extension association. This action doesn't delete extensions defined in the association.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/appconfig-2019-10-09/DeleteExtensionAssociation>`_  


  **Request Syntax**
  ::

    response = client.delete_extension_association(
        ExtensionAssociationId='string'
    )
    
  :type ExtensionAssociationId: string
  :param ExtensionAssociationId: **[REQUIRED]** 

    The ID of the extension association to delete.

    

  
  
  :returns: None
