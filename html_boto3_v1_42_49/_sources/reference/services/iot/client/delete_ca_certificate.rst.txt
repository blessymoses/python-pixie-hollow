delete_ca_certificate

.. py:method:: IoT.Client.delete_ca_certificate(**kwargs)

  

  Deletes a registered CA certificate.

   

  Requires permission to access the `DeleteCACertificate <https://docs.aws.amazon.com/service-authorization/latest/reference/list_awsiot.html#awsiot-actions-as-permissions>`__ action.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iot-2015-05-28/DeleteCACertificate>`_  


  **Request Syntax**
  ::

    response = client.delete_ca_certificate(
        certificateId='string'
    )
    
  :type certificateId: string
  :param certificateId: **[REQUIRED]** 

    The ID of the certificate to delete. (The last part of the certificate ARN contains the certificate ID.)

    

  
  
  :rtype: dict
  :returns:
