.. py:method:: IoT.Client.delete_mitigation_action(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iot-2015-05-28/DeleteMitigationAction>`_

        actionName='string'

  :type actionName: string
  :param actionName: **[REQUIRED]**
