list_sbom_validation_results

.. py:method:: IoT.Client.list_sbom_validation_results(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/iot-2015-05-28/ListSbomValidationResults>`_
