.. py:method:: MailManager.Client.stop_archive_search(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/mailmanager-2023-10-17/StopArchiveSearch>`_

  ::

    response = client.stop_archive_search(
        SearchId='string'
    )
    
  :type SearchId: string
  :param SearchId: **[REQUIRED]** 

    The identifier of the search job to stop.

    

  
  
  :rtype: dict
  :returns:
