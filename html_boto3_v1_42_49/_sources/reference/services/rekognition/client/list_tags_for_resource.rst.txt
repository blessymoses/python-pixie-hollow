.. py:method:: Rekognition.Client.list_tags_for_resource(**kwargs)

  

  Returns a list of tags in an Amazon Rekognition collection, stream processor, or Custom Labels model.

   

  This operation requires permissions to perform the ``rekognition:ListTagsForResource`` action.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rekognition-2016-06-27/ListTagsForResource>`_  


  **Request Syntax**
  ::

    response = client.list_tags_for_resource(
        ResourceArn='string'
    )
    
  :type ResourceArn: string
  :param ResourceArn: **[REQUIRED]** 

    Amazon Resource Name (ARN) of the model, collection, or stream processor that contains the tags that you want a list of.
