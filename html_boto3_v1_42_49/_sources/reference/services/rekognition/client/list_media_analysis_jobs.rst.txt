list_media_analysis_jobs
************************



.. py:method:: Rekognition.Client.list_media_analysis_jobs(**kwargs)

  

  Returns a list of media analysis jobs. Results are sorted by ``CreationTimestamp`` in descending order.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rekognition-2016-06-27/ListMediaAnalysisJobs>`_  


  **Request Syntax**
  ::

    response = client.list_media_analysis_jobs(
        NextToken='string',
        MaxResults=123
    )
    
  :type NextToken: string
  :param NextToken: 

    Pagination token, if the previous response was incomplete.

    

  
  :type MaxResults: integer
  :param MaxResults: 

    The maximum number of results to return per paginated call. The largest value user can specify is 100. If user specifies a value greater than 100, an ``InvalidParameterException`` error occurs. The default value is 100.

    

  
  
  :rtype: dict
  :returns:
