tag_resource
************



.. py:method:: Rekognition.Client.tag_resource(**kwargs)

  

  Adds one or more key-value tags to an Amazon Rekognition collection, stream processor, or Custom Labels model. For more information, see `Tagging AWS Resources <https://docs.aws.amazon.com/general/latest/gr/aws_tagging.html>`__.

   

  This operation requires permissions to perform the ``rekognition:TagResource`` action.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/rekognition-2016-06-27/TagResource>`_  


  **Request Syntax**
  ::

    response = client.tag_resource(
        ResourceArn='string',
        Tags={
            'string': 'string'
        }
    )
    
  :type ResourceArn: string
  :param ResourceArn: **[REQUIRED]** 

    Amazon Resource Name (ARN) of the model, collection, or stream processor that you want to assign the tags to.

    

  
  :type Tags: dict
  :param Tags: **[REQUIRED]** 

    The key-value tags to assign to the resource.
