describe_flow_execution_records

.. py:method:: Appflow.Client.describe_flow_execution_records(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/appflow-2020-08-23/DescribeFlowExecutionRecords>`_  


  **Request Syntax**
  ::

    response = client.describe_flow_execution_records(
        flowName='string',
        maxResults=123,
        nextToken='string'
    )
    
  :type flowName: string
  :param flowName: **[REQUIRED]** 

    The specified name of the flow. Spaces are not allowed. Use underscores (_) or hyphens (-) only.

    

  
  :type maxResults: integer
  :param maxResults: 

    Specifies the maximum number of items that should be returned in the result set. The default for ``maxResults`` is 20 (for all paginated API operations).

    

  
  :type nextToken: string
  :param nextToken:
