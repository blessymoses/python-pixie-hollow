.. py:method:: Appflow.Client.untag_resource(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/appflow-2020-08-23/UntagResource>`_  


  **Request Syntax**

    response = client.untag_resource(
        resourceArn='string',
        tagKeys=[
            'string',
        ]
    )
    
  :type resourceArn: string
  :param resourceArn: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the flow that you want to untag.

    

  
  :type tagKeys: list
  :param tagKeys: **[REQUIRED]**
