  .. py:method:: wait(**kwargs)

    Polls :py:meth:`PrometheusService.Client.describe_anomaly_detector` every 2 seconds until a successful state is reached. An error is raised after 60 failed checks.

    See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/amp-2020-08-01/DescribeAnomalyDetector>`_    


    **Request Syntax**
    ::

      waiter.wait(
          workspaceId='string',
          anomalyDetectorId='string',
          WaiterConfig={
              'Delay': 123,
              'MaxAttempts': 123
          }
      )
      
    :type workspaceId: string
    :param workspaceId: **[REQUIRED]** 

      The identifier of the workspace containing the anomaly detector.

      

    
    :type anomalyDetectorId: string
    :param anomalyDetectorId: **[REQUIRED]** 

      The identifier of the anomaly detector to describe.

      

    
    :type WaiterConfig: dict
    :param WaiterConfig: 

      A dictionary that provides parameters to control waiting behavior.

      

    
      - **Delay** *(integer) --* 

        The amount of time in seconds to wait between attempts. Default: 2

        

      
      - **MaxAttempts** *(integer) --* 

        The maximum number of attempts to be made. Default: 60

        

      
    
    
    :returns: None
