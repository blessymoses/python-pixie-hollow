*****************
describe_document
*****************



.. py:method:: SSM.Client.describe_document(**kwargs)

  

  Describes the specified Amazon Web Services Systems Manager document (SSM document).

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/ssm-2014-11-06/DescribeDocument>`_  


  **Request Syntax**
  ::

    response = client.describe_document(
        Name='string',
        DocumentVersion='string',
        VersionName='string'
    )
    
  :type Name: string
  :param Name: **[REQUIRED]** 

    The name of the SSM document.

     

    .. note::

      

      If you're calling a shared SSM document from a different Amazon Web Services account, ``Name`` is the full Amazon Resource Name (ARN) of the document.

      

    

  
  :type DocumentVersion: string
  :param DocumentVersion: 

    The document version for which you want information. Can be a specific version or the default version.

    

  
  :type VersionName: string
  :param VersionName: 

    An optional field specifying the version of the artifact associated with the document. For example, 12.6. This value is unique across all versions of a document, and can't be changed.

    

  
  
  :rtype: dict
  :returns:
