**************************
deactivate_key_signing_key
**************************



.. py:method:: Route53.Client.deactivate_key_signing_key(**kwargs)

  

  Deactivates a key-signing key (KSK) so that it will not be used for signing by DNSSEC. This operation changes the KSK status to ``INACTIVE``.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/route53-2013-04-01/DeactivateKeySigningKey>`_  


  **Request Syntax**
  ::

    response = client.deactivate_key_signing_key(
        HostedZoneId='string',
        Name='string'
    )
    
  :type HostedZoneId: string
  :param HostedZoneId: **[REQUIRED]** 

    A unique string used to identify a hosted zone.

    

  
  :type Name: string
  :param Name: **[REQUIRED]** 

    A string used to identify a key-signing key (KSK).

    

  
  
  :rtype: dict
  :returns:
