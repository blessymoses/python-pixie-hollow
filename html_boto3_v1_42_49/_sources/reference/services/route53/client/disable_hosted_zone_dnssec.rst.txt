disable_hosted_zone_dnssec
**************************



.. py:method:: Route53.Client.disable_hosted_zone_dnssec(**kwargs)

  

  Disables DNSSEC signing in a specific hosted zone. This action does not deactivate any key-signing keys (KSKs) that are active in the hosted zone.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/route53-2013-04-01/DisableHostedZoneDNSSEC>`_  


  **Request Syntax**
  ::

    response = client.disable_hosted_zone_dnssec(
        HostedZoneId='string'
    )
    
  :type HostedZoneId: string
  :param HostedZoneId: **[REQUIRED]** 

    A unique string used to identify a hosted zone.

    

  
  
  :rtype: dict
  :returns:
