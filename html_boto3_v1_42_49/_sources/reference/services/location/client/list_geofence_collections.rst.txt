list_geofence_collections

.. py:method:: LocationService.Client.list_geofence_collections(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/location-2020-11-19/ListGeofenceCollections>`_  


  **Request Syntax**
  ::

    response = client.list_geofence_collections(
        MaxResults=123,
        NextToken='string'
    )
    
  :type MaxResults: integer
  :param MaxResults: 

    An optional limit for the number of resources returned in a single call.

     

    Default value: ``100``

    

  
  :type NextToken: string
  :param NextToken: 

    The pagination token specifying which page of results to return in the response. If no token is provided, the default page is the first page.

     

    Default value: ``null``
