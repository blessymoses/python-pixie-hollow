.. py:method:: DeviceFarm.Client.get_test_grid_project(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/devicefarm-2015-06-23/GetTestGridProject>`_

        projectArn='string'
