.. py:method:: QApps.Client.untag_resource(**kwargs)

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/qapps-2023-11-27/UntagResource>`_  


  **Request Syntax**
  ::

    response = client.untag_resource(
        resourceARN='string',
        tagKeys=[
            'string',
        ]
    )
    
  :type resourceARN: string
  :param resourceARN: **[REQUIRED]** 

    The Amazon Resource Name (ARN) of the resource to disassociate the tag from.

    

  
  :type tagKeys: list
  :param tagKeys: **[REQUIRED]**
