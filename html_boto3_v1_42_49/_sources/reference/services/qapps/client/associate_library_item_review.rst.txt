associate_library_item_review
*****************************



.. py:method:: QApps.Client.associate_library_item_review(**kwargs)

  

  Associates a rating or review for a library item with the user submitting the request. This increments the rating count for the specified library item.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/qapps-2023-11-27/AssociateLibraryItemReview>`_  


  **Request Syntax**
  ::

    response = client.associate_library_item_review(
        instanceId='string',
        libraryItemId='string'
    )
    
  :type instanceId: string
  :param instanceId: **[REQUIRED]** 

    The unique identifier for the Amazon Q Business application environment instance.

    

  
  :type libraryItemId: string
  :param libraryItemId: **[REQUIRED]** 

    The unique identifier of the library item to associate the review with.

    

  
  
  :returns: None
