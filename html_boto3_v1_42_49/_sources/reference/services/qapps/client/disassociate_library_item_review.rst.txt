disassociate_library_item_review
********************************



.. py:method:: QApps.Client.disassociate_library_item_review(**kwargs)

  

  Removes a rating or review previously submitted by the user for a library item.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/qapps-2023-11-27/DisassociateLibraryItemReview>`_  


  **Request Syntax**
  ::

    response = client.disassociate_library_item_review(
        instanceId='string',
        libraryItemId='string'
    )
    
  :type instanceId: string
  :param instanceId: **[REQUIRED]** 

    The unique identifier of the Amazon Q Business application environment instance.

    

  
  :type libraryItemId: string
  :param libraryItemId: **[REQUIRED]** 

    The unique identifier of the library item to remove the review from.
