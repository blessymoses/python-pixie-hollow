******************
stop_q_app_session
******************



.. py:method:: QApps.Client.stop_q_app_session(**kwargs)

  

  Stops an active session for an Amazon Q App.This deletes all data related to the session and makes it invalid for future uses. The results of the session will be persisted as part of the conversation.

  

  See also: `AWS API Documentation <https://docs.aws.amazon.com/goto/WebAPI/qapps-2023-11-27/StopQAppSession>`_  


  **Request Syntax**
  ::

    response = client.stop_q_app_session(
        instanceId='string',
        sessionId='string'
    )
    
  :type instanceId: string
  :param instanceId: **[REQUIRED]** 

    The unique identifier of the Amazon Q Business application environment instance.

    

  
  :type sessionId: string
  :param sessionId: **[REQUIRED]** 

    The unique identifier of the Q App session to stop.

    

  
  
  :returns: None
